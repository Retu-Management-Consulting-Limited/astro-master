# Molly 关系引擎 · STATUS（living，更新于 2026-06-16）

## 一句话诊断
Molly 承诺「一个比你自己更懂你的人」，但现状是**参数表**不是**关系**。缺三样：**记忆**（越来越懂你）、**主动**（牵着你走）、**specificity**（说的是你，不是你的星座）。

## 第一原则（凌驾一切）· R18（`DESIGN-SYSTEM.md §16`）
**抓住人性 = 服务，不是剥削。** ① 狠是赚来的（随信任）② 盟友非审判（权威给星盘）③ 给掌控不逼墙角 ④ 门=认知诚实非 usage gate ⑤ 日活靠"有人记得你"非 streak 威胁 ⑥ 永远落到能动不留伤口。

## 架构：一个地基 + 六个出口
```
心情 / 说中了吗 / 聊过什么 → ⟨UserModel 真实"懂你"记忆⟩ → 渲染六个出口
                                     ↑ keystone：狠度·深度·解锁 全是它的函数
```
