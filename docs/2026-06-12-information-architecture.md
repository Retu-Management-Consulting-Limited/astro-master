## 1. 进场漏斗（首次 · 转化关键路径）

📕 小红书链接 → 🛬 落地（先抛痛点、不问星座）→ 📝 输入**出生日期 + 出生时间（不知道→校准小测）+ 出生地（国家+城市，供 geocoding + 历史时区）**（**输入页不放登录**）→ 🎯 **校准小测**（埋上升 + 第一刀被看穿）→ ✨ **首读（激活峰）** → 🔑 **注册留存**（先体验后注册；Email 首选/Google/Apple）→ ⭐ **加到主屏（PWA，峰值触发；召回命门）** → 进入「今日」

## 2. 底部四 tab

### ① 今日（home）
昨（验证·喂模型）/ 今（指引·一刀一糖）/ 明（钩子·明日相位）。＝习惯回路 + 召回落点。详见 [每日仪式](2026-06-12-daily-ritual.md)。

### ② 本命
命盘图（[盘面美学](2026-06-12-chart-visual-design.md)，高亮驱动）+ 主题导览（亮点偵測排序）+ 四痛域深度解读（🔒 未来付费单位，占位不设计）。＝深度池。
- 💰 **财运日历**（[详见](2026-06-12-wealth-calendar.md)）：从本命独立出来、给**显眼入口**（强留存件，首屏可达）。绿→白→红整月网格。

### ③ 对话
追问 Iris / 遇事问她（决策伴侣）/ C 毒舌·自适应语气（[大师人设](2026-06-12-master-persona.md)）。＝粘性核心 + 真实建模主入口。（tab 名「对话」；曾用旧 working name「一眼」，已随大师改名 Iris 弃用）

### ④ 我的
校准度 & 自我模型（护城河可视化）/ 历史回看 🕰️ / 我的金句卡 / 合盘入口 💞 / 设置（含 AI 揭露）。

## 3. 贯穿全局的两条暗线

- 🕳️ **潜伏式校准**：在四个 tab 的每次互动背后偷偷收窄模型（喂模型 = 一鱼两吃）。
- 📨 **召回**：Email + Web Push + PWA 把人勾回「今日」。
