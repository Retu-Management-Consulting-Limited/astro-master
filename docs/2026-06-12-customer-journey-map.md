# 星盘大师 · 全客户旅程地图

> **文档类型**：客户体验战略 / 旅程地图
> **状态**：设计进行中（brainstorm 阶段产出，非最终 spec）
> **日期**：2026-06-12
> **适用产品**：星盘大师（AI 西洋占星本命盘 Web App）

---

## 0. 产品背景（截至本文档的设计共识）

让本文件可独立阅读。截至目前，产品设计已确认以下主线：

- **核心体验**：AI 大师解读对话 ＋ 精准排盘。既要算得准的西洋本命盘，也要 AI 大师用白话解读、能追问。
- **占星体系**：西洋占星本命盘（演算用成熟开源天文历库，如 Swiss Ephemeris）。
- **企图**：走向商业化，但 **MVP 先不做付费转化**，专注"前期人性收割机"——把使用与传播做透。
- **互动模式**：**C 引导式骨架 ＋ B 追问对话**。左侧主题导览（解掉"空白页问题"、切出未来付费单位），每个主题内嵌聊天框可追问；进场有一段 A 式"开场速读"当钩子。
- **核心差异化引擎 ①——亮点偵測**：先让 AI 扫过命盘、抓出对此人最突出/最戏剧性的 2–3 个配置，据此个人化生成主题顺序与追问 chips。「同一个 app，每个人看到的引导都不一样」。
- **核心差异化引擎 ②——潜伏式校准**：后台维护"出生时间机率分布"，从校准小测的宽窗口开始，把每次互动（准/不准反馈、聊天吐露的人生大事、伪装成"让解读更贴"的微问题、行为信号）当证据，贝氏式收窄、漸进逼近分钟级。**越用越准 ＝ 留存 + 护城河 + 召回弹药**。
- **目标客群**：精准在"痛点"、宽在人群。锁四大高频痛域——**两性关系 / 事业财富 / 身心健康 / 自我方向**。
- **时代定位（AI 世代）**：冥王星进水瓶（2024–2043）＝占星学现成的"科技颠覆 / AI 革命"世代象徵。定位金句：「当 AI 让一切变得不确定，回到你的星盘，找回 AI 拿不走、只属于你的人生坐标。」用 AI 交付"对抗 AI 焦虑"的确定感。
- **核心原则——戳准 + 接住**：痛点一定要戳、往死里戳准（"被说中"是多巴胺来源），但戳完那句必须是"我懂你"的释放，不是"你完了"的恐惧。**精准戳中服务"被看见"，不服务"被吓到"**；传播靠其他维度（合盘、身分卡、金句），与痛点引擎分开长。
- **传播机制（待细化）**：C1 社交货币（日月升卡片）、C2 合盘病毒回路、C3 截图金句。

## 1. 情绪曲线总览

低位好奇（被痛点轻钩）→ 锯齿状防备（期待与"怕被割韭菜"拉扯）→ **急升至第一个大高峰「激活峰」**（校准的"咦准了" + 首读的"被看见"震撼）→ 高位震荡的前 7 天（每次被说中是小高潮，但有新鲜感衰减的下行压力）→ 活跃期的规律小高潮波浪（形成情感依赖）→ **再升至「暖峰」**（分享/合盘的兴奋与归属）→ 流失期的危险下行深谷（新鲜耗尽，或痛点疲劳导致焦虑反噬）。

> **整条曲线最该守的，是不让活跃期那条波浪线被"越戳越焦虑"拖进深谷。**

```
情绪
 高 │              ★激活峰              ★暖峰
    │             /‾\               /‾\
    │            /   \___      ____/   \
    │   _      _/        ‾‾\__/         \
    │  / \____/                          \
 低 │_/                                   \____ ⚠深谷
    └────────────────────────────────────────────
     认知  考虑  决策/激活  上手7天  活跃  忠诚  流失
```

## 2. 分阶段旅程地图

### ① 认知 Awareness — *情绪：低位好奇 + 一丝怀疑*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 深夜刷到"准到爆"的星盘截图 / 看到"AI 抢工作"新闻后焦虑搜索 → 点开。想："这好像在说我""真的假的"。**触发认知的不是"想算命"，而是一个痛点时刻**（刚分手、怕被裁、生日前的存在焦虑）。 |
| **触点** | 数字＝社媒 feed（小红书/抖音/IG/TikTok）、KOL 占星内容、朋友转发的合盘结果、SEO（"上升星座查询""冥王水瓶 事业""AI 会取代我吗"）、应用商店。实体＝朋友面对面安利、"你什么星座"的日常话题。 |
| **痛点/摩擦** | "又一个骗钱占星 app"的预设怀疑；占星 app 同质化；落地页若泛泛，3 秒划走。 |
| **惊喜机会点** | 落地页第一眼就个人化——不问星座，先抛"你最近是不是在为某件事反复纠结？"；用 AI 世代定位差异化（不是"今日运势"，而是"AI 时代你拿不走的人生坐标"）。 |
| **关键指标** | 曝光→点击 CTR、UGC referral 占比、目标搜索词覆盖/排名、落地页跳出率、CAC。 |
| **优化工具/战术** | KOL 种草、UGC 金句卡裂变（C3）、蹭"AI+占星"热词的 SEO/短视频、落地页 A/B、归因追踪。 |

### ② 考虑 Consideration — *情绪：略升但呈锯齿，期待与防备并存*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 看评价、看示例解读、试着输入资料。想："值得花时间吗""会不会看一半逼我付费""我连出生时间都不知道"。怕被割韭菜。 |
| **触点** | 应用商店评价、落地页示例解读、社媒评论区口碑、产品试用首屏。 |
| **痛点/摩擦** | **"不知道出生时间"＝最大劝退点**（普通 app 这里直接卡死）；注册墙太早；索取过多个人信息引发隐私戒心。 |
| **惊喜机会点** | **校准小测把"不知道时间"从劝退变卖点**（"不知道也没关系，我们帮你校准到分钟"）；零注册先给速读再谈注册；用一段"被看见"的示例击碎"罐头占星"预期。 |
| **关键指标** | 落地页→开始校准转化率、示例解读互动率、注册前流失点、"不知道出生时间"用户的留存对比。 |
| **优化工具/战术** | 先体验后注册、示例个性化、隐私信任信号（数据怎么用、可删除）、校准小测当破冰、社会证明（"X 万人已校准"）。 |

### ③ 决策 / 激活 Conversion — *情绪：★ 急升至第一个大高峰* *(付费已 defer，转化＝完成首读并产生首次追问)*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 完成校准 → 读首读 → 第一次追问。想："哇这真的懂我""我还想知道更多"。从防备→惊艳→卸下心防。**第一个情绪峰，留存命门。** |
| **触点** | 渐进式校准流程、首读结果页、第一个追问对话框。 |
| **痛点/摩擦** | 校准题太多会累；**首读若泛泛＝致命**；加载卡顿在此刻杀伤最大。 |
| **惊喜机会点** | 首读必埋"起鸡皮疙瘩金句"（C3）；结尾用亮点偵測抛出"为你挑的"追问，顺势带进对话；可视化校准度（"已校准 78%"）给即时成就感。 |
| **关键指标（最关键阶段）** | 校准完成率、首读读完率、**首次追问率（核心激活动作）**、首读→D1 次留。 |
| **优化工具/战术** | 渐进式校准、首读模板 + 人工 QA 打磨金句、aha 时刻设计（＝第一次被说中）、即时反馈回路（校准度上升动效）。 |

### ④ 上手 Onboarding · 前 7 天 — *情绪：高位震荡，小高潮叠加衰减压力*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 回访、探索更多痛域、看每日星象、继续追问。想："今天它会说什么""我的事业宫到底怎样"。逐渐生出期待与依赖。 |
| **触点** | push 通知（每日星象 / AI 时事扣星象）、逐步解锁主题、追问对话、"校准度提升"提示。 |
| **痛点/摩擦** | 内容重复→兴奋衰减；push 过频→反感卸载；**第 2–3 次解读若没有新的"被看见"＝流失**；空白页。 |
| **惊喜机会点** | 第 N 天"**你的盘校准更准了，重新看看你的感情宫**"制造二次惊艳（潜伏引擎复利变留存武器）；AI 时事扣星象的及时内容；亮点偵測持续推新主题解决空白页。 |
| **关键指标** | D1/D3/D7 留存、7 日内 session 数、每日星象打开率、主题探索广度、**"第二次被看见"达成率**、push opt-in & CTR。 |
| **优化工具/战术** | 7 天 onboarding 序列编排、push 策略（少而准、扣时事）、新鲜度引擎（行运＋校准提升＋AI 时事）、固定每日触发时刻养习惯。 |

### ⑤ 活跃 Engagement — *情绪：规律小高潮的稳定波浪线*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 日常打开、遇人生大事来追问（"我要不要接这个 offer""该不该转行做 AI"）、随手反馈准不准。**"遇事问大师"成为条件反射**。 |
| **触点** | 每日星象、事件驱动的深度追问、反馈按钮、校准度里程碑（潜伏引擎背后持续收窄）。 |
| **痛点/摩擦** | ⚠️ **高频痛点被反复戳却没"接住"→焦虑积累/疲劳**（护栏失守、滑向深谷的第一推力）；解读趋同；缺乏进展感。 |
| **惊喜机会点** | 把人生大事追问做成「**决策伴侣**」（用户真把它当遇事咨询对象＝超级粘）；校准里程碑庆祝；AI 时事专题。 |
| **关键指标** | WAU/MAU、人均 session、追问频次、**"遇事追问"占比（深度激活）**、反馈率、校准度增长曲线、⚠️ 情绪健康信号（投诉/负面反馈率）。 |
| **优化工具/战术** | 事件触发内容、决策伴侣定位、里程碑庆祝、**情绪护栏监测（戳准但要接住）**、个性化推送节奏。 |

### ⑥ 忠诚 Loyalty / Advocacy — *情绪：★ 再升至暖峰，兴奋与归属*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 截图金句发群、拉另一半/朋友合盘、晒日月升卡、口头安利。想："这太准了你也来""我们合盘看看"。 |
| **触点** | 分享卡生成、合盘邀请链接、截图金句、社媒、朋友间。 |
| **痛点/摩擦** | 分享物不够美/不够"想曬"→不分享；合盘要求对方也注册→摩擦；缺自然分享触发时机。 |
| **惊喜机会点** | **合盘做成病毒回路**（一次分享＝一个新用户，C2，最强裂变）；分享卡极致美学（C1）；**在情绪峰值自动抛分享时机**（刚读到震撼金句时弹"分享这句给懂你的人"）。 |
| **关键指标** | K 因子/病毒系数、分享率、合盘邀请发出 & 转化、referral 占新增比、NPS、UGC 产量。 |
| **优化工具/战术** | 合盘病毒机制、精美分享卡、金句截图优化、克制的推荐激励（别破坏"自发"感）、峰值时机触发分享。 |

### ⑦ 流失 Churn — *情绪：危险下行，冷却 / 厌倦 / 焦虑反噬*

| 维度 | 内容 |
|---|---|
| **动作/想法/情绪** | 减少打开、关 push、卸载。想："都差不多""没新东西""有点烦"。情绪冷却，或被累积焦虑反噬。 |
| **触点** | 沉默期、push、流失后召回邮件/push。 |
| **离开原因** | ① 新鲜感耗尽、解读趋同 ② **痛点被反复戳却没接住→焦虑疲劳（护栏失守）** ③ 校准度停滞→"越用越准"承诺破灭 ④ push 过度 ⑤ 痛点已解决/转折期已过（自然流失）。 |
| **早期预警信号** | 打开频次下降、追问停止、反馈停止、push CTR 跳水、session 变短、解锁/探索归零。 |
| **惊喜机会点** | 潜伏引擎精准召回（"你的盘又校准更准了，回来看重新校准的你"）；行运/人生新事件触发召回（"你最近有一道重要转运"）；退订/卸载前提示"你的校准会归零"（损失趋避）。 |
| **关键指标** | churn 率、复活率、**预警命中率**、沉默→流失转化时长、召回 CTR、⚠️ 情绪健康/投诉率（护栏指标）。 |
| **优化工具/战术** | 流失预警模型（行为信号驱动）、事件/行运触发召回、情绪护栏（监测"越用越焦虑"）、价值提醒挽留。 |

## 3. 战略综合：3 个最高杠杆时刻 + 1 个最大风险

1. **激活峰（首读"被看见"）** — 整条曲线的命门。校准没让人"咦准了"、首读没让人起鸡皮疙瘩，后面全崩。值得投入最多打磨与人工 QA。
2. **暖峰的合盘病毒回路** — 唯一能让 K 因子 >1、把 CAC 压下来的杠杆。一次分享＝一个新用户，必须极致顺滑。
3. **潜伏式校准的复利** — 同时是留存（越用越准）、护城河（切换归零）、召回弹药（重新校准）。贯穿全旅程的暗线引擎。
4. **⚠️ 最大风险 ＝ 活跃期的"焦虑反噬"** — 刻意戳高频痛点换多巴胺，但"戳准"没配"接住"，会把忠诚用户一路推下流失深谷。**"情绪健康率"必须是和留存并列的北极星护栏指标**，不能只看 DAU。
