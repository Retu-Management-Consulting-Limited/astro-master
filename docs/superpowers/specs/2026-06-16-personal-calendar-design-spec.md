## A. 今日格（做实）

### A.0 三态共用骨架（同一套槽，按态填）

| 槽位 | 作用 | 红·慎 | 绿·宜/旺 | 平淡·天清 |
|---|---|---|---|---|
| ① 顶条 | 日期 + 状态标 | 日期 · 慎（红点） | 日期 · 宜（绿点）/旺 | 日期 · 天清 |
| ② 主句（领头·命令） | 直白下令（规则8） | 「慎 · 今天别{冲动动作}」 | 「宜 · 今天去{可掌控动作}」 | 「今天天上没大动静」 |
| ③ 兴奋层（可选·可晒） | 前向诱饵（规则11） | — | 「旺 · 搞钱黄金日」+轻免责 | — |
| ④ 为什么（内在） | 钉在她内在态（规则8,I1） | 「你现在判断在情绪上头」 | 「今天你说得出口、压得住场」 | 「没有谁来拨你的弦」 |
| ⑤ 糖（赦免） | 戳完给糖（规则9） | 「不是他有问题，是月亮刑你金星」 | （并入④的底气，不外归因结果） | 「适合把攒的情绪落一落」 |
| ⑥ 动作 / 门 | 落到可控（规则10,11） | 一扇前向门「留到周四再说」 | 一个动作「报价抬一档」 | （平淡日无强动作） |
| ⑦ 微互动（回溯验证） | 喂模型，只问内在（规则12） | 「昨天那个慎，你忍住了吗」 | 「上一个宜日，你动了吗」动了/怂了 | 备战糖那道题即验证 |
| ⑧ 备战糖（平淡专属·可选） | 歇之上叠加（规则14） | — | — | 认信号/提前备/练一招 三选一 |

### A.1 红 · 慎（刹车）

- **领头**：命令叫停。「慎 · 今天别要答案 / 别下这单」
- **为什么**：必须内在、可自证。「你现在判断在情绪上头」——不写"今天会出事"（外部，违 I1）。
- **糖（赦免）**：把责任推给天象。「不是你的错，是月亮刑你金星」。
- **门（规则10 硬性）**：永远留前向出口。「真想说的，留到周四——那天话是你的，不是情绪的」。**禁止**「你不行/别动」式关门。
- **微互动**：次日此格变"昨天"时问「昨天那个慎，你忍住了吗 👍👎」。**绝不问**「昨天亏了吗」。
- **稀有**：红有月度配额（建议 ≤ 3–4 天/月），无命中不强升红（规则13 平淡为底）。

### A.2 绿 · 宜/旺（勇气义肢 / 油门）

- **领头**：命令出手。「宜 · 今天去开口谈」。
- **兴奋层（可选）**：标题可披「旺 · 搞钱黄金日」做社交货币，**必带轻免责**「旺的是时机，不是保证」。
- **为什么（内在勇气）**：「今天你说得出口、压得住场」——**不写**「今天会赚钱/胜率高」当正式承诺（外部可证伪）。
- **动作**：一个她可掌控的真实动作 + 微助推（开口谈 / 收欠款 / 报价抬一档）。
- **微互动**：「上一个宜日，你动了吗 —— 动了 / 怂了」。量勇气义肢有没有起效（规则12 + D）。
- **顺序律（规则6 / I4）**：信任未攒够的早期，**少推绿、多用红攒信任**；绿是花信任，得先有信任可花。

### A.3 平淡 · 天清（多数日）

- **先给歇（规则13）**：「今天天上没大动静，难得一天平。没有谁来拨你的弦。」赦免，别造戏。
- **再叠备战糖（规则14，连续平淡到第 3 天升起；可选，想接才接）**，三型：
  1. **认信号**（教她读自己）：「你发现没——你最容易冲动花钱的日子，月亮都在你财帛宫。认住它。」
  2. **提前备**（指向已知红绿日）：「周四是你这月'宜开口'的日子，这两天先把数字、底线想好。」
  3. **练一招**（绑她盘的小习惯）：「情绪上头先压一晚再回消息——你这种金星天蝎最该练这条。」
- **命门**：必须**关于她**（非通用占星知识，否则成罐头）、**一口像礼物**（非作业）、**看人给**（陈昊爱②、林星爱①、小漾秦姐多半只要"歇"+一句暖）。
- **白赚**：备战糖常是一道题，她答 = 喂模型（规则14 + D：提高她与喂模型同一招）。

### A.4 文案硬约束（每条都来自规则/I）

1. 主句是**命令**不是观察（规则8）。
2. ④为什么 = 内在态；⑦验证 = 内在/行为。外部事件**只能**出现在③兴奋层且带免责（I1/I2a）。
3. 红必留门（规则10）；绿必落可控动作（规则11）。
4. 一格之内**净情绪为正/为安**（规则19 / I8）——红也要靠④⑤⑥让她走时是被接住、不是被吓。

## B. 决策引擎 → 今日格 的接口（B 必须吐这个对象，A 才渲染得出）

A 不关心 B 怎么算，只消费这个**每人每日一个**的对象：

```
DailyVerdict {
  date
  state:        '红' | '绿' | '平淡'
  channel:      '钱' | '健康'            // 今天哪条轨在说话
  mode:         'house' | 'planet'        // 时辰可信→宫位增强；不可信→降级行星相位
  intensity:    1..3                     // 配额/稀有度用
  leadAspect:   '月刑金' | '木合二宫' ... // 领头相位（给文案）
  natalHit:     她盘上被命中的点          // 个性化锚
  lean:         '刹' | '推'              // 表型决定（规则5）
  innerState:   '情绪上头' | '有底气' ... // 填④为什么（内在词）
  sugar:        '月亮刑你金星' ...        // 填⑤赦免（天象归因）
  action/door:  可控动作 或 前向门 + 目标日 // 填⑥
  nextCharged:  下一个红/绿日的日期        // 红日的门、平淡日的提前备 都要它
  prepCard?:    认信号/提前备/练一招 候选   // 平淡日的备战糖
}
```

### B.1 采访锁定（6 决策）

| 决策 | 选择 |
|---|---|
| 大脑 | 混合：规则定骨架 + LLM 写文案 |
| 稀有 | 天定为主 + 保底配额 |
| v1 范围 | **钱 + 健康 双轨**（B 每天多判一层"哪轨说话"）|
| 表型起手 | 盘推（火/土）+ 行为学修正 |
| 时辰 | **C 双模式**（house-enhanced / planet-degraded），**校准为前提** |
| 冷启动 | 复用 daily-ritual「5 天钩子弧」，B 约 Day5 接棒 |

### B.2 两层结构

- **B1 判定层**（确定性规则 × 真星历）：算出骨架 `state/channel/mode/lean/intensity/leadAspect/natalHit/nextCharged`。可测、守动态内容契约（R14–R15）。
- **B2 文案层**（LLM）：拿 B1 骨架 + 她的自我模型，按 Molly 声音 + charter 文案硬约束（命令式 / 内在 why / 赦免糖 / 红留门 / 验证问内在）写 `innerState/sugar/action·door/prepCard`。**出口过 I1 检查**：禁止 LLM 私自塞外部事件预测。

### B.3 B1 判定六步

1. **命中检测**：今日行运行星 → 本命点位的相位。`mode=house` 纳入 2/8/6 宫与四角；`mode=planet` 只用行星↔行星（跳过宫位/上升）。按 orb 紧密度排序（最紧=最定义她，沿用亮点偵測）。
2. **选轨**：每个命中映射 channel——钱（金/木/土 → 2/8 宫主或财星）/ 健康（月/火/土 → 月亮/6 宫/上升）。最紧命中的轨今天说话；软性防单调：同一轨连说 N 天，让另一轨的次紧命中浮出。
3. **定 state**：相位性质定红绿——扩张/和谐（木/金，拱/六分）→ 绿候选；收紧/张力（土/火，刑/冲；月火、月土 = 情绪上头）→ 红候选；无紧密命中 → 平淡。
4. **叠表型（规则5）**：冲动型 → 张力相位重判红、绿更谨慎给（规则6 早期少推）；胆怯型 → 扩张相位重判绿、她需要被推。**同相位按表型渲染不同**。
5. **卡稀有（天定+保底）**：orb 够紧才升红绿；月度封顶（红 ≤~4/月）；长期全平则保底把"现有最强那天"提为红绿，防整月没戏；约两周内若有候选必给 ≥1 绿（油门不缺席）。
6. **算 intensity + nextCharged + prepCard**：intensity 由 orb + 行星权重定（驱动文案"手感"档，叠表型）；前扫下一个红绿日（红的门、平淡的提前备要它）；平淡且连续 ≥3 天 → 按表型选 prepCard 型（陈昊→提前备 / 林星→认信号 / 小漾秦姐→轻或免）。

### B.4 B × D 耦合（校准是前提）

B 的输入不是"时辰有/无"二值，是 **D 持续收窄的「时辰可信度分布」**。B 按当前可信度选 `mode` 与宫头精度：分布宽 → planet 降级；收窄 → house 增强；中间连续过渡。**D 校准 → B 自动从行星中心升级到宫位中心。B 与 D 是一个环，不是先后两块。**

**校准三条诚实注脚（碰了会塌）：**
1. **冷启动头 ~2 周校准还没收窄，B 的 planet 模式必须独立扛住前两周**——不能拿"以后会准"当借口（恰是 5 天弧最脆处）。
2. **机制上"校准" = 拟合她不断点头的那个自己**（自我模型/护城河），不是天文意义找回真时辰。要**保留被看穿的棱角**，别只迎合，否则那口冷汗就没了。
3. **最强校准信号是行为 + 人生事件（生时校正 rectification），不是"准/不准"按钮**（噪声大、有讨好偏差）。D 据此给信号加权。
