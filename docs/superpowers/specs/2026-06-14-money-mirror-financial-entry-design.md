## 0. 一句话

> **Molly 用「钱对你到底意味着什么」当钩尖，把你的金钱故事一天天讲给你听——越讲越准、从不让你难堪。钱是天天回来的留存脊柱，爱情是炸出去的病毒尖峰；金融分发，是这根脊柱够粗以后的第二步。**

把 Molly 零散的「财运」内容，升级成一个**连载式的金钱自我叙事**：钩尖照见钱对你这个人意味着什么（安全/地位/自由/价值），日引擎用希望主调的故事把你天天勾回来，地板是「被看见却不被评判」的信任。这是 Molly **赚得资格**日后当金融产品入口的地基。

## 1. 战略判断

### 1.1 为什么是「钱」当核心钩子

钱是现代生活的**万能溶剂**——它同时代理生存、地位、自由、自我价值、对未来的掌控五样核心焦虑。别的话题各管一格，钱一格压五格。作为**钩子**，它比爱情有两个结构性优势：

- **更宽**：钱的焦虑不分单身已婚、不分男女老幼，人人都有；失恋只勾正在失恋的人。
- **更日常**：人每天都在跟钱发生情绪（每笔消费、看价格、发薪日、「我配不配」、刷到别人过得好）。这种**环境级、永不结案**的焦虑，是比爱情戏剧好得多的**留存基质**。

### 1.2 双钩分工（定稿）

| 钩子 | 角色 | 触发 | 优势 |
|---|---|---|---|
| **钱** | 🦴 留存脊柱 | 日常、环境级 | 宽、高频、留得住 |
| **爱情/关系** | 🚀 病毒尖峰 | 失恋/暧昧/情绪尖峰 | 炸点、传得开（K 因子，喂小漾） |

钱负责天天回来，爱情负责炸出去。**别让钱去抢爱情的尖峰活**，也别指望爱情扛日留存。

### 1.3 拒绝「财运日历直接导流交易」的 naive funnel

最直觉的「今天财运旺 → 按钮 → 去交易」机制上是坏的：

> **占星最容易转化成「立刻做个金钱动作」的用户，恰恰是最差的金融客户；真正的金融买家不会被这种方式转化。**

成立的桥梁是**行为/意义桥**（「钱对你意味着什么」「你为情绪买单」），干净映射到合法、低风险的金钱行为，而非投机。

### 1.4 三条路与分阶

| | A · 核心钩子地基（本 spec） | B · 稳健理财/保险 分发 | C · 交易/CFD |
|---|---|---|---|
| 收入 | 中（订阅/深读） | 中高（佣金） | 最高（即时） |
| 风险 | 低 | 中 | **极高** |
| 能否当 wedge | ✅ 地基 | ✅ A 之后 | ❌ 不能 |

**Path C 判断**：零售 CFD 用「运气」框架、瞄准情绪用户、跳过适当性测试，正是触发误导销售/脆弱客户规则的东西，**不能当 wedge**。**Path B 是第二步**，待 A 的留存脊柱被验证、信任与金钱处境数据成型后再谈；本 spec 只埋接缝。

> 重述命题：Molly 通过**先用钱当钩子勾住并被信任**，赚得日后当金融入口的资格。

## 2. 第一性原理：钱面前的人性

### 2.1 六条公理

1. **没人要钱，人要钱的含义**——安全、被看得起、自由、被爱、「我够格」。钱是代理变量。最深的钩子不是「你会发财」，是 **「钱对你到底意味着什么」**。
2. **人受不了「命不知道」，也受不了「命定死了」**。甜点是 **「命里有这个可能（被看穿，安心）＋ 钥匙在你手上（有得选，赋能）」**，两者必须同时给。
3. **最深的快感是「被看见，且不被评判」**——被看穿是最稀缺的情感品。
4. **钱的主情绪按人群分裂**（v1 误判：羞耻不是普世主情绪，只是一个寄存器）：
   - 匮乏/求安者（秦姐、小漾）→ 羞耻 + 恐惧 + 不配得。
   - 进取/地位者（陈昊）→ **怕掉队 + 争胜 + 掌控**，不羞耻。
   - 自由/意义者（小漾/林星）→ 不被困住、活出自己。
   占星的神机关对羞耻段尤其有效：**让星盘背锅，让人留住尊严**（「不是你的错，是你土星的位置」）；但对争胜段无效——他要的是掌控/领先。
5. **人凭情绪行动，再用逻辑给自己台阶**。哪怕「理性」的陈昊也按「怕被淘汰」买、用 ROI 包装。
6. **身份 > 行为**。人不为「管好花钱」改变，会为「成为账户终于听话的那种人」改变。

### 2.2 钩尖：「钱对你到底意味着什么」

因为公理 4 分裂，钩尖**不能**是单一情绪（v1 的「卸羞耻」只接住秦姐/小漾，把陈昊推开）。钩尖必须是一个**随人变形**的镜片：

> **「Molly 知道钱对你到底意味着什么。」**

- 秦姐 → 钱 = 安全 + 照顾家人
- 陈昊 → 钱 = 地位 + 掌控 + 不被时代甩下
- 小漾 → 钱 = 自由 + 我配得上
- 林星 → 钱 = 不被定义、活出自己的方式

同一个钩子，钩尖随人变形——对羞耻的人说「不丢人」，对争胜的人说「我知道你怕掉队」。**这才戳中大部分人，而非一段人。** 钩尖**星盘播种、行为精化**（见 §4 `behavior.ts`）：Molly 越用越懂钱对你意味着什么——「越用越准」终于在长一个真东西。

### 2.3 勾住 = 三段力

| 力 | 作用 | 落点 |
|---|---|---|
| **被看见**（钩尖刺入） | 第一次「卧槽好准」的激活尖峰 | S1 揭示。**一次性、会衰减——不能当日引擎** |
| **尊严/不评判**（钩不脱钩） | 让用户敢一直待着、敢说真话 | 护栏 + 见证回路。信任地板 |
| **连载希望叙事**（每天回来的引擎） | 可再生、永不结案的「我的故事在推进」 | 见 §3。**日留存的真引擎** |

### 2.4 产品本质

> **Molly 在钱这件事上的工作，是当那个唯一「知道钱对你意味着什么、看见你却不评判你、还把你的金钱故事一天天讲下去」的存在。** 财运是表皮，**被懂 + 尊严 + 希望叙事**才是引擎。

### 2.5 一把必须守住的刀（伦理 = 策略）

「钱的羞耻/恐惧」可卸也可捅，捅短期转化更高，但捅 = 红线明禁的掠夺，且烧掉策略赖以成立的长期信任。**本设计死守「治愈」侧**——这刚好也是唯一能长出可变现信任的那一侧。

## 3. 日引擎：连载希望叙事（含预言层）

日留存最深的基质不是「希望」本身，是**叙事**——人每天回来，是为了一个关于自己、正在展开的故事的下一页。希望是情绪主色，**叙事是结构**。

### 3.1 三个机制

1. **连载**：「你的金钱故事」每天有新一页，有承前启后（昨天/今天/这段时间），不是孤立的今日运势。
2. **希望主调 + 真实天气对比**：纯希望会腻、会滑向画饼。日内容由**真实行运**塑形——有旺有慎，**护住式的「慎」给对比**，让「旺」显得是挣来的、真的。没有对比，希望就是噪音。
3. **money-meaning 染色**：同一套引擎，按钩尖（§2.2）给每天上色——对陈昊染成「今天是你拉开身位的窗口」，对秦姐染成「这段适合给家里筑一道更稳的墙」。**钩尖不是日内容，是给日内容上色的滤镜。**

### 3.2 预言：大胆预言，不报数字

人性要预言（公理 1、2），但有唯一一道悬崖：**可证伪的「金额 × 日期」**（「你明年赚 50 万」）。它会被现实核对、常错 → 自毁准度感知（留存命门）；且预言收益 +（日后）产品推荐 = 误导销售。**沿四条 trust-safe 轴大胆预言**：

- **窗口（时机）**：「9–11 月你最旺的扩张窗口开着」——方向性、真实（木星行运确是扩张窗口）、非金额。
- **命运（人格）**：「你这辈子的钱靠一次敢转向」——像预言、不可证伪。**去赌性**（不用「敢赌」）。
- **条件式（if-then，最强）**：「这半年你守住情绪化消费，年底大概率看到一个你没见过的数字」——给数字的**感觉**、系在用户自己的行为上；「if 你守住」本身就是行为桥。**成了庆祝、没成不追责**（删甩锅暗面，否则制造愧疚）。
- **量级质感**：「不是小数目，是够你喘口气的一笔」——给质感不给数字。

**定位**：「预言体验、不预言可证伪的金额」。在 §4 落成类型层硬约束。

## 4. 架构与单元

套 Molly 现有模式（纯 `lib/` 确定性计算 + AI 写文案围绕注入事实 + `/api` 路由 + KV/local 持久）。新金钱域置于 `web/src/lib/money/`。

| 单元 | 类型 | 职责 | 接现有 |
|---|---|---|---|
| `persona.ts` | 纯函数 (TDD) | ChartFacts → `MoneyPersona{meaning, strengths[], blindSpot, styleTag}` —— **`meaning` 是钩尖轴（安全/地位/自由/价值），星盘播种** | `lib/astro` 星位（金/木/火/土 + 2/8 宫） |
| `narrative.ts` | 纯函数 (TDD) | (Persona, transits, horizon, lastChapters) → `Chapter{tone, hopeNote, prophecy, tiedToMeaning}` —— **连载 + 希望主调 + money-meaning 染色** | 行运计算；预言只能产 window/destiny/conditional/texture |
| `nudge.ts` | 纯函数 (TDD) | (Persona, dayTransits) → `DayNudge{tone, action}` —— 护住式「慎」给对比 | **升级 `lib/astro/wealth.ts`** |
| `guardrail.ts` | 纯函数 (TDD) | `validateMoneyCopy()`：六公理硬门 + 免责 + 脆弱软化 | 红线集中地 |
| `profile.ts` | 状态 | 持久 `MoneyProfile{persona, chapterLog, behaviorLog, consent, riskFlags}` | KV/local；**即金融接缝数据体** |
| `behavior.ts` | 纯+状态 (TDD) | 见证回应 → 精化 `meaning` + 算准度 → refinePersona | **激活 stub 的 self-model；钩尖「越用越准」靠它** |
| `seam.ts` | 接口（A 不实现） | `getReferralContext()` flag-off 返回 null + intent 日志 | 定义 B 契约，零外部调用 |

### 4.1 落在类型层的硬防护

- **金额 × 日期 = 编译期不可表达**：`Chapter.prophecy` 里**根本没有** `{amount, date}` 字段，只能产 window/destiny/conditional/texture。
- **`guardrail.ts` 是所有金钱文案的唯一出口**：结构化禁词扫描（货币数字+未来日期 / 羞耻句式 / 赌性 = 硬 BLOCK）+ 强制轻免责 + 脆弱软化。

### 4.2 付费深读

走现有 `/api/reading` 加 `moneyBlueprint` 模式（接 `TODO(key)` 付费门）。AI 写文案，persona + narrative 注入确定性事实（「模型不许编星位/金额」）。**注：付费件在 Slice 0 不建（见 §9）。**

## 5. 变现与定价（不捅羞耻地赚钱）

> 注：变现是 A 验证（§9 Slice 0）**之后**的事，此处定方向。

**三个掏钱时刻（都不靠恐惧）**：被看见后想看得更深（好奇/身份）；想把懂我的人留在身边（关系延续）；希望峰值的冲动（非恐惧峰值）。

| 层 | 内容 | 定价 |
|---|---|---|
| **免费**（钩子） | 钩尖揭示 + 每日希望叙事 + 基础预言窗口 | ¥0，信任地基，绝不收费 |
| **金句卡**（传播） | 「我的搞钱命格」可晒卡 | ¥9–19（小漾，喂 K，套 `/share`） |
| **金钱蓝图深读**（主变现） | 陈昊级完整报告 | ¥68–198（陈昊/林星，接 `TODO(key)`） |
| **Molly 在·订阅**（留存） | 持续叙事 + 推送 + 见证 | ¥30/月；¥30 年度全家（秦姐 LTV） |

**变现红线（定价层保险丝）**：

> **付费墙只能立在「更深 / 更久 / 更趁手」前面，绝不立在「安心」前面。** 免费层必须永远能让用户被看见、且觉得自己会没事地离开。

延伸到 B：佣金同此红线——「你现在配得上一个更稳的位置」（尊严/希望），不是「你这样会穷死」（羞耻）。

## 6. 测试（把人性钉成可执行的门）

1. **纯逻辑 TDD**：`persona.ts` 同盘→同人格（含 meaning 轴确定性）；`narrative.ts` 断言结构里**不存在**金额+日期、连载承前（引用 lastChapters）；`nudge.ts` 行运→对的红/白/绿；`guardrail.ts` 「赚 50 万」/「你会更穷」/「敢赌」→ BLOCK。
2. **guardrail = 伦理写进 CI**：六公理禁区做成反例语料库，全部接住。
3. **LLM-as-judge 评语气**（改造 `agent-test-runner`）：样本盘生成叙事 → 六公理 rubric 打分：被看见？卸羞耻/给掌控（按 meaning）？有希望？给钥匙？冒数字/赌性？**+ 跨人群：换 meaning 轴是否真的换了情绪寄存器（陈昊版≠秦姐版）。**
4. **E2E**：钩尖揭示 → 日叙事 → 见证回路；**断言付费墙从不挡免费「安心」内容**。
5. **真人金丝雀**：林星（专业及格线）+ 秦姐（护栏生死线）。

## 8. 风险与缓解（执行风险已升为头号）

| 风险 | 级别 | 缓解 |
|---|---|---|
| **内容罐头化**：日叙事撑不到第 30 天，重复/翻车（林星「又一个罐头」） | 🔴 **头号** | §9 Slice 0 先验保鲜度；行运塑形 + meaning 染色 + 连载给无限变化；LLM-judge 评重复 |
| 钩尖只接住一段人（争胜段被推开） | 🔴 | meaning 轴随人变形；Slice 0 必含陈昊型金丝雀 |
| 日引擎押在「被看见」（一次性）→ 留存空 | 🟠 | 引擎改连载希望叙事，被看见只做激活尖峰 |
| 语气滑向教练/说教 | 🟠 | 身份锁「懂你的人」；希望打头；LLM-judge |
| 见证回路逼用户认错 | 🟠 | 零评判、失败也接住；优先间接推断 |
| 预言制造愧疚/赌徒 | 🟠 | 条件式不追责；去赌性；类型层禁数字×日期 |
| 变现捅羞耻 | 🟡 | 付费墙红线，E2E 硬断言 |
| self-model 仍 stub | 🟡 | `behavior.ts` 真正回灌 meaning 精化 |
