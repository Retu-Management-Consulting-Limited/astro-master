## 0. 一句话定性（产品魂）

> **个人日历不卖"未来会怎样"的断言，卖"今天扛得住 + 明天值得期待"——让她一天一天、不那么怕地走过去，把不确定变得可承受、可下一步、有个真盼头、有人陪。**

它不是预言机，是**勇气义肢**：松开她本来就想做的那件事的刹车（或，对上头的人，踩下她本来该停的那一脚）。卖的不是运势，是一颗**瞄准行动、不是瞄准感觉**的安慰剂。

## 规则（白话定论版 · 这是定论；下面各节是它的机制依据/精确版）

**一、先立心：这是什么**
1. 这是一台"今天机器"，不是算命。它**卖今天的可承受 + 明天的盼头**——盼头是"未来**值得期待**"（黄金日/行运高峰，真时效，§7 绿），不是"未来**会怎样**"的断言（"你会赚 X"那种，编、会翻车，§8 红）。落点永远是：让她今天敢迈一步、且有个值得朝它倾斜的明天。
2. 人真正扛不住的是：未来太大、太不确定。日历的本事就是把它切成"一次只过一天"。
3. 别去降低不确定（占星没这本事，一试就被拆穿），只把不确定**变得能扛**。不预测，只接住。

**二、它怎么起效**
4. 它是一颗安慰剂，但瞄准**行动**，不瞄准**感觉**。决定早在她心里，缺的只是勇气——它松刹车。
5. 给上头的人踩刹车，给胆怯的人松勇气。同一句话，看人调方向。
6. **先刹车攒信任，再用油门花信任。** 先靠"说中你正上头"让她信，再推她行动。顺序反了推不动。
7. 刹车比油门值钱：冲动决策多半是错的，"今天别动"靠概率就对一大半，躲过的坑还记它的功。

**三、每天那一格怎么写**
8. **直白下命令，别温柔绕弯。**"慎·今天别要答案"比"你心里那根弦容易被拨动"有力十倍。笃定才有效。
9. **一刀一糖**：戳完立刻给糖。糖不是温柔，是赦免——"不是你的错，是今天的天象"。
10. **叫停必留门**：红日不说"你不行"，说"今天先别，留到周四"。叫停不能把人钉死。
11. 前向标题可以兴奋、可以晒（"搞钱黄金日"），但落点必须是**她自己能做的一个动作**。
12. 回头只问"**你做了吗**"，永远不问"赚了吗/中了吗"。问她能自证又骗不了的，别问会翻车的。
13. **平淡日先给"歇"**：今天天清、没大事压你，先赦免她。别天天硬造戏——天天有戏＝没戏，红绿就不值钱。
14. **平淡日在"歇"之上再递一口"备战糖"**（想接的接、不接就歇）：一个**关于她自己、又指向即将到来红绿日**的小东西——①认信号（教她读自己）②提前备（指向已知红绿日）③练一招（绑她盘的小习惯），让她觉得"今天没白来、还更准备好一点"。命门：必须**关于她**（非通用知识，否则成罐头）、**一口像礼物**（非作业）、**看人给**（陈昊爱备数字、林星爱认信号、小漾秦姐多半只要歇+暖）。这一口顺手把数据喂了模型——"提高她"和"喂模型"是同一招。

**四、日历长什么样**
15. **今天是家**：一打开就是今天那一格，零选择，直接挨那一击。
16. 月历、章节是家里的房间，能直接走到，**但不是三个并排的门**。别让"今天"沦为三选一之一。
17. **月历是伪装成氛围图的规划器**：一眼扫这个月疏密（多数平、偶尔红绿），点未来某天能看那天该干嘛。氛围留住秦姐，规划卖给陈昊。
18. **章节（这一程）只在大转折来时才浮出来**，平时收着。别挂一个几个月不变的死 tab。

**五、红线（碰了就死）**
19. **情绪强度拉满，但只能建在真实星象/真实事件上。** 戳痛、一针见血、诚实的紧迫感——全放开（总宪法 §3.3/§7 绿灯）；唯一禁区＝**无中生有或刻意夸大负面情绪当杠杆**（§8.1 红）。真痛可以，编的痛不行。§9 安全（危机/脆弱/数据）永不松。
20. 不怂恿赌博投机、不卖招财改运、不"不转发破财"。财运是占星最容易吃人的地方，守死。
21. **建模透明可查、但不喧宾夺主。** 设置页如实揭露 AI 与"越用越准"怎么运作（总宪法 §6.3），**不欺骗、不误导她关于真实的事（§8.3）**；日常前景是"它懂我"的爽，不是"它在算我"的冷——这是不隐瞒下的取舍，不是藏死。每用一个信号，当场还她一个真好处。

> 规则与下面 I1–I8 是同一件事的两个高度：白话规则是定论与对外口径，I1–I8 是精确判定与违宪审查依据。冲突时以精确版的机制为准校准白话，不改变白话结论。

## 1. 真需求（第一性原理推导）

1. 人对未来的痛，本质不是"信息缺失"，而是**未来是一团无限的、无法一次性面对的迷雾**。占星对未来没有因果通道（核因果机制为空集），所以产品**不可能**降低这份不确定——一旦尝试，就是预言、可证伪、会死。
2. 真正终结不确定之苦的，从来不是拿到缺的信息，是**面对未知的关系变了**：交托、信则灵、被接住。需求不是"弥补/填缺口"，是**"让不确定变得可承受"**。
3. 满足只发生在**她的内在姿态**层（安抚情绪 / 解冻一个下一步 / 给章节感 / 陪着），**永不**发生在外部事件信息层。
4. **日历是这个需求的原生几何**：它把无限迷雾切成有名字的格子，让她**一次只面对一天**，把"远方的恐惧"预消化成"明天那一格的下一步"。这就是为什么形态必须是日历，而非聊天机器人或报告。

## 2. 机制：安慰剂 = 勇气义肢 / 刹车（同一引擎，双向）

- **决定早就在，缺的是勇气。**"明天是你出手的好日子"不预测任何未来，它松开她那一脚的刹车 → 她行动 → 预言**通过她自己的行动**自我兑现。活性成分是**勇气/行动**，不是结果。
- **同一台引擎朝相反方向开**：
  - **给上头的人踩刹车**（过度行动者：陈昊 FOMO 乱交易、小漾凌晨逼问）。
  - **给胆怯的人松刹车**（行动不足者：秦姐从不为自己做事）。
- **刹车 > 油门 的信任性价比**（三条第一性）：① 冲动型决策 base rate 多半错，"今天别动"靠基础概率就对了大半；② 归因——避坑的功劳记 app，赢钱的功劳被她自己截胡；③ 损失厌恶 ~2x，"帮我躲过一个错"比"帮我赢一把"重一倍。

## 3. 不可违反的不变量（宪法条款）

> 任何 UI / 算法 / 文案上线前，逐条过。违反任一条 = 违宪，打回。

- **I1 · 内在不外部，验态不验果。**
  所有"准不准 / 说中了吗"指向的，必须是**她此刻可自证的内在体验/状态**（"昨天你是不是手痒想立刻下手"），**绝不**是外部事件结果（"昨天是不是赚到钱 / 这单会不会亏"）。外部 claim 可证伪，一次落空，安慰剂**永久失效**。信任**不押在反事实上**，押在"它说中了我当时上头/上心"上。
  **澄清（2026-06-16 修订）**：本条只管**回溯的验证钩**。前向的**判决/标题**走另一把尺（见 I2a）——宇宙式命令/性质（"今天慎 / 旺"，不可即时证伪、且把责任推给天象＝赦免）**可以**外部；只有前向的**事件预测**（"会收到坏消息 / 今天会亏 / 会赚 X"）禁止。线不在"内在 vs 外部"，在"命令/性质（不可证伪）vs 事件预测（可证伪）"，加上"验证钩专留内在态"。

- **I2 · 安慰剂三药理。**
  - (a) **活性成分是勇气/行动，不是结果。** 前向的**今日标题/钩子**可以披一层有节制、可晒的外部兴奋（"财运旺 / 搞钱黄金日"——它是社交货币与传播燃料，配轻免责"旺的是时机不是保证"，靠确认偏误与赦免存活），**但正文必须落到一个她可掌控的真实动作**；而**回溯的验证钩永远只问内在/行为**（"昨天你去开口/去要了吗"），**绝不问外部结果**（"赚到了吗"）。验证是唯一被审计到现实的地方，只审她能自证、又正好量到勇气有没有起效的东西。**一句话：兴奋放前向的诱饵里，严谨放回溯的审计里。**
  - (b) **稀缺保护药效** → 不能天天好日子/天天警告。大多数日子必须真·平淡；**空白格子是洗脱期，不是留存 bug，不许被填满**。
  - (c) **治疗剂量 ≠ 商业剂量** → 必须分两层、不许混：
    - **每日定向仪式（维生素）**：便宜、低强度、平淡日也开（"今天什么调子"）——撑日活。
    - **稀有的勇气/刹车干预（药）**：一个月就那么几天真·按住或真·推动——撑改变。

- **I3 · 红日前门律。**
  红日（刹车日）只能给**保护性许可 + 一扇前向出口**（"今天先别拍板，留到周四那个属于你的日子"），**绝不**给关上的门（"你不行，别动"）。否则勇气义肢反向长成把人钉在原地的**怯懦义肢**。locus of control 永远锁在内部："今天你会怎么运作"，不是"会发生在你身上的事"。

- **I4 · 刹车赚信任，油门花信任（顺序律）。**
  先用"说中你正上头"的刹车建立可信（高 base rate、损失厌恶、内在可自证、可归因）；**再**拿攒下的可信去花在稀有绿日的勇气一推。信任没建就先催行动 = 推不动。

- **I5 · 表型配比律。**
  刹车/油门的配比按用户**行为表型**（冲动 vs 怯懦）走，不按"通用运势"。早期用盘面起手（火/火星重→偏刹车；土星重/胆怯→偏勇气），之后靠"偷偷变准"的自我模型从她的真实行为学（对绿日动不动手、对红日听不听、悔的是冲动还是错过）。**个性化真正该长在"该踩她还是该推她"，不是个性化运势文案。** 两向都必须稀而高信念，杜绝 cry-wolf。

- **I6 · 时间方向律（沿用核心原则 §4）。**
  验证永远回溯，预测永远前瞻。任何"验证"指向的事必须已发生且她此刻可观测。「昨/今/明」= 功能三分（昨=验证喂模型 / 今=指引 / 明=钩子），非时间装饰。快图层（月亮/内行星）撑日常、慢图层（外行星）撑章节，不混用。

- **I7 · 记忆从属律。**
  记忆（过去）本身**不是需求**，唯一价值是给"今天那句会没事的"**充信任电**，使前向的勇气/刹车可信到她真能歇一口气。不服务前向的记忆 = 虚荣功能，删。

- **I8 · 真而不编（红线，重写以从属总宪法 §8）。**
  情绪强度 / 紧迫感 / 留存钩 **全拉满**——只要建在真实星象、真实事件上（§7 绿）。唯一禁区＝**无中生有或夸大负面情绪当杠杆 · 靠编造的坏后果/愧疚召回 · 靠欺骗换指标**（§8 三亮线）。仍守：不怂恿赌博/投机、不卖招财改运、不"不转发破财"（§8.2 红）。§9 安全（危机/脆弱人群/敏感数据）永不松。**留存要狠，靠的是"让她真的离不开被你看懂的爽"（总宪法 §10），不是靠编造的吓唬。**

## 4. 双轨内容（钱 + 健康），感情外置

日历脊柱 = **两条真·每日不确定轴**：

| 轨 | 占星底料 | 它对她是 | 主味 |
|---|---|---|---|
| **钱（财运）** | 二宫/八宫、金星/木星/火星/土星行运 | "今天该不该*做*这一步" | **推一步 / 刹车**（agency） |
| **健康（身心/能量）** | **月亮（每日情绪天气，已是快图层引擎）** + 六宫 + 火/土行运 | "今天你的容量/状态，能不能硬扛" | **安抚 / 陪着**（permission） |

- 一刚一柔：钱管"做"，健康管"在"，合起来盖住两条普世每日不确定。
- **健康轨**（月亮快图层底料已在手）守"真vs编 + §6.4/§9"，线就一条＝**自证性**（同全产品的 I1：验态不验果、只问她答得了的）：**她自己答得了的，可问、可喂模型；她答不了的，不可点名。**
  - ✅ **倾向**：她怎么待自己的真实模式——硬扛/透支/憋累/忽略信号（真、可自证、驱动自我照顾，§4.4 情感诚实，**有分量必说**）。
  - ✅ **把"可能的身体信号"当假设抛给她、让她拿真实症状自证**："月亮压你六宫，是不是睡眠/肠胃先替你喊累？" 她答对/不对 ＝ **被看穿 + 喂模型**，逐步攒出她的"身体压力点"（累先从哪冒）＝ **真校准、越用越准**。
  - ✅ **自证出的症状有分量 → 点"这块值得留意" + 推她看真医生**（§6.4 转专业、§9 安全）。app 是把她送去真正能测的人那里的推手。
  - ❌ **她答不了的（病种/诊断）不可点名/断言**："你是不是有甲状腺/心脏病"——她靠体感答不了（要靠检查），问了只会逼她把普通症状（心慌其实是焦虑）错挂到吓人的病上＝nocebo；占星也没命名具体病的依据（编、冒充医疗 §6.4、撞 §9 硬机器）。
  - 一句话：**症状让她自证、攒成"了解她"；病留给医生。**
- **感情 = 事件触发的病毒钩，活在日历之外**（危机点燃、可晒、合盘可分享）——负责拉新（K 因子），不负责每日留存。日历不是感情的家。

## 5. 违宪审查清单（任何屏/流/文案上线前）

1. **I1**：这句反馈/验证指向的是她可自证的**内在态**，还是可证伪的**外部果**？（外部 → 改）
2. **I2(a)**：这句的承诺落在"她靠行动能兑现"的地带，还是"会发生在她身上"的地带？
3. **I2(b)**：今天若是平淡日，这个格子是被尊重为洗脱期，还是被硬塞了戏？
4. **I2(c)**：这是每日维生素层，还是稀有药物层？两层混了没？
5. **I3**：红日有没有留一扇前向出口？还是关上了门？
6. **I4**：在信任还没攒够时，是不是就开始催她行动了？
7. **I5**：这是按她的行为表型在踩/推，还是发通用运势？刹车/油门够不够稀、够不够高信念？
8. **I6**：时间方向对吗（验证回溯、预测前瞻）？快慢图层混没混？
9. **I8**：这一屏让她净安还是净焦？一个月累计呢？

> 你不做"回答未来"的生意（会死），你做"让不知道变得安全、让她敢迈下一步"的生意。

## 附录 A · 今日格三态（2026-06-16 设计定案）

「那一格 15 秒」经可视化碰撞，定下三态。**直白命令领头胜过温柔叙事壳**（理由：笃定＝放大安慰剂剂量；命令＞观察＝更高行动力与权威；宇宙判决＝把责任推给天象＝赦免）。

- **红 · 慎（刹车）**：判决式命令领头（红灯 +「慎 · 今天别要答案」）→ 内在 why（「你现在判断在情绪上头」）→ **前门**（「留到周四，那天话是你的，不是情绪的」）→ 内在验证钩。守 I1/I3/I4/I5。
- **绿 · 宜/旺（勇气义肢）**：命令领头（「宜 / 旺」）→ 标题**可披"财运旺 / 搞钱黄金日"的可晒外部兴奋**（配轻免责，作社交货币）→ **正文落到一个可掌控动作**（开口谈 / 收欠款 / 报价抬一档）→ **验证钩只问"你动了吗"，绝不问"赚了吗"**。守修订后 I1/I2a/I4。
- **无灯日（多数日）**：默认 **①留白呼吸**（诚实「今天没大事」＝喘息，守 I2b 洗脱期）；**连续平淡到第 3 天自动切 ③维护日**（喂自我模型 + 提前埋一个绿日盼头）。**禁止 ②天天榨"小天气"**——那是测测/Co-Star "a loop not a journey" 的罐头死法。

> 三态共识：**兴奋放前向、严谨放回溯；红稀、绿稀、平淡为底；命令直白，糖是赦免不是柔软。**
