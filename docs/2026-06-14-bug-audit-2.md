## P2 — polish / a11y / edge (selected; full set in agent outputs)

- **♅ wrong glyph for ascendant** ([me/page.tsx:39](web/src/app/me/page.tsx#L39)) — ♅ is Uranus, not Asc; reads as an error to astrology-literate users.
- **No truncation** on nickname/email ([me:38](web/src/app/me/page.tsx#L38), [settings:137](web/src/app/me/settings/page.tsx#L137)) — long values overflow the header/row.
- **`window.prompt()` for nickname edit** ([settings:73](web/src/app/me/settings/page.tsx#L73)) — jarring native dialog, suppressed in some PWA webviews.
- **Unconsented gender default** ([me/birth/page.tsx:28](web/src/app/me/birth/page.tsx#L28)) — `?? "female"` persists "female" to a pre-feature user's profile on first save, feeding the gendered AI tone.
- **Fire-and-forget `apiSync`** ([me/birth:47](web/src/app/me/birth/page.tsx#L47)) — silent server-sync failure; user thinks the edit saved.
- **Frozen prefilled birth defaults** ([input/page.tsx:23](web/src/app/input/page.tsx#L23)) — `1998-06-13`/墨尔本; tap-through submits a fake chart.
- **chat `aria-live` wraps the whole list** — re-announces prior messages on each send.
- **maximumScale:1** ([layout.tsx:21](web/src/app/layout.tsx#L21)) — blocks pinch-zoom (WCAG 1.4.4).
- **VAPID subject default** `mailto:hello@vapeincity.com` ([push.ts:33](web/src/lib/server/push.ts#L33)) — likely-wrong contact baked as fallback.
- **notification badge** uses full-color `/icon.svg` ([sw.js:91](web/public/sw.js#L91)) — Android badges need a monochrome silhouette.
- **share channel buttons** (微信/小红书/Instagram) all call `exportPng` ([share:98](web/src/app/share/page.tsx#L98)) — only「保存图片」is name-accurate. (Prior SH-1, unfixed.)
- **No `<h1>`** on landing; **calibration ritual + reading 9.99M-ms loader** edge cases; **register guest path skips push opt-in**; **anniversary-recall push promise** ([history:57](web/src/app/history/page.tsx#L57)) unbacked.
