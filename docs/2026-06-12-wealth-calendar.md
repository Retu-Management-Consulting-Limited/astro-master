## 1. 双重价值（不只是「她们在乎」）

- 🔁 **日历＝回访机器**：天天来看「今天搞钱行不行」，天然的每日复访格式 → **直打头号死因（留存）**。
- 📅 **规划价值**：「本月搞钱黄金日」让她安排谈薪/签约/大额消费 → 高频实用，有人专门来查。
- 📤 **可晒**：「我的搞钱黄金日」＝社交货币，喂小红书传播。

## 2. 视觉：绿→白→红

- **越旺越绿 / 平用白 / 越慎越红**，整月网格 + 底部渐层图例。
- 红绿灯语义＝**行动灯**（绿=去做、红=停手），与「旺去做事·慎别乱动」对齐。
- 注：中文股市红涨绿跌相反，但此处是「行动灯」非「涨跌灯」，绿旺红慎正确。

## 3. 文案规则（定稿）

- **旺（绿）**：「财运旺」+「今天投资的胜率也偏高」+ 主推**她能掌控的真实金钱动作**（谈薪、报价、收欠款、推合作）。轻免责：「旺的是时机，不是保证——功课别省，决定你来做。」
- **慎（红）**：「**不适合任何投资和大额消费决定**」+ Iris 的护住式提醒（「不是坏日子，是提醒你今天的钱跟着情绪走」）。
- **页脚轻免责**（保护气氛也保护项目）：「财运仅供参考 · 投资有风险，决定还是你做」。
- **昨日验证守规则**：问内在/行为（「昨天那个慎，你忍住了吗」），不问「昨天赚到钱了吗」（外部/翻车）。

## 4. 两条红线

1. **行动时机，非财富预测**（守 P2 镜不预言）：说「适合/避免做什么」「胜率偏高」，不说「你会赚 X / 财运 90 分」这种可证伪的硬预测。
2. **赋能，非贩卖焦虑/掠夺**：帮她抓时机、避冲动；**不慫恿赌博（彩票/投机）**、不「不转发破财」、不卖招财改运——财运正是占星最易掠夺之处。

## 5. 内容来源与个人化

- 扎在她真实的**金钱宫位行运**（二宫、八宫、金星、木星=扩张机会、火星=冲动消费、土星=收紧），行运天天变 ＝ 无限内容。
- **必须经「懂你」个人化**（结合自我模型/亮点偵測：「你这种金星天蝎，财运的敌人是为情绪买单」）——这是与测测「商品化今日财运」的差异化所在。

从「本命」独立出来、给显眼入口（强留存件，值得首屏可达）。
