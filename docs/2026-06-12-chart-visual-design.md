## 1. 定调：前台审美物件，不是占星师工作底稿

传统占星软件的星盘＝工程图（密密麻麻符号/度数/相位线），对小红书女性又丑又劝退。

> 我们要做的不是「准确的星盘图」，是**「她想盯着看、想晒出去的、属于她的宇宙」**。盘面＝前台审美物件 + 互动地图；精密计算藏后台。皮肤要美，器官藏起来。（呼应「占星是介面、AI 自我建模是引擎」）

## 2. 美学方向：A + D + 微 B（已定）

- **A 极简现代**（骨架）：留白、稀疏、premium，像真盘但被极简化 → **可信，过林星关**。
- **D 东方金质感**（皮肤）：藏蓝 + 金 + 东方纹样（如旋转方块/几何），西占混东方质感 → 高级感、差异化、呼应华人/离散身份，且＝现有品牌色。
- **B 星空微光**（点缀）：星点 + 星云微光 + 高亮发光 → 梦幻、想晒。
- **不采 C 治愈插画为主**：太软，会稀释「C 毒舌通透」的人设气质。

## 3. 三个产品层决策（比颜色更重要，已定）

1. **高亮驱动，不 show 全部**：默认只让**亮点偵測**抓出的 2–3 个配置发光突出，其余星暗化。盘面＝她的重点，不是天文清单。
2. **互动地图，非静态图**：点发光的星 → 跳出该块解读，把盘面接进「主题导览」。盘是入口。
3. **越用越亮**：随校准度上升，盘面视觉逐渐清晰/发光更强——把「越用越准」变成**看得见**（P6 护城河可视化）。

## 4. 社交货币

一张美盘本身即社交货币（C1）：可单独「把我的星盘存成图」分享，与金句卡并列为传播资产。盘面顶部展示「校准度 X%·盘面会越来越亮」，既是进度感也是回访钩子。

## 5. 待设计 / 技术
