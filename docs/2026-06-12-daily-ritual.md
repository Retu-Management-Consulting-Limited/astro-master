# 星盘大师 · 每日仪式（昨/今/明）

> **文档类型**：功能设计 / 留存引擎
> **状态**：设计进行中（brainstorm 阶段产出）
> **日期**：2026-06-12
> **配套**：[核心设计原则](2026-06-12-core-design-principles.md)（§4 时间不变量）
> **目标**：让用户每天花约 5 分钟，内容永不枯竭

---

## 1. 核心认知：占星是内容供给天然无限的品类

天上的星每天都在动（**行运 transits**）：每天不同行星划过她出生盘的不同位置，形成不同角度。「今天对你意味着什么」**逐日不同、逐人不同、且算出来不用人写**。内容是程序化生成（ephemeris × 她的盘 × AI 渲染），不靠编辑团队。

## 2. 「昨/今/明」三段＝三个功能（非时间装饰）

> 时间不变量：**验证永远回溯，预测永远前瞻。**（缘起：曾误把「准不准」放进未结束的「今天」——未发生之事无法验证。）

| 段 | 占星机制 | 功能 |
|---|---|---|
| **昨天** | 昨日相位已过、结果已知 → 可验证 | **验证**：喂模型 + 坐实「准」 |
| **今天** | 今日生效的相位 | **指引**：前瞻、可行动、带金句（一刀一糖）|
| **明天** | 明日即将精确的相位（真实天象）| **钩子**：留存 + 损失趋避，明日 push 依据 |

自转的轮子：今天的指引，明天自动变成被验证的「昨天」——验证永不缺素材。

**两条专业细节**：
1. **验证问内在体验，不问外部事件**：「昨天你是不是特别想抓住一个确定答案？」（内在、易命中、低证伪风险），不问「昨天是不是收到好消息？」（外部、易落空、咒语一破即失）。
2. **验证回路＝金矿**：把「行运 × 她的盘」的推测拿去对账她真实生活体验，是唯一非巴纳姆的真实数据，让模型一天比一天准。

## 3. 5 分钟「今日」屏四段流

1. **今日天气（15 秒，召回钩子）**：一句话个性化（「今天月亮刑你的金星，感情别急着要答案」）。
2. **今日为你（1–2 分）**：扣今日相位、落在她此刻最痛那一域的解读 + 金句。
3. **微互动（1 分）**：「昨天那件事说中了吗？」👍👎 / 心情打卡（表面参与，里子喂模型）。
4. **钩子（1 分）**：问大师 / 明日相位预告 / 存成卡片。

## 4. 快慢分层（同一时间轴两个图层）

- 🌙 **快图层**（月亮/内行星，日内—数日）：撑「昨/今/明」日常循环。
- 🪐 **慢图层**（外行星，数月—数年）：撑「你这阵子正在经历一个大转折」章节叙事——冥王水瓶＝AI 世代定位的落点。

## 5. 内容供给七源（永不枯竭）

行运（主引擎）· 月亮（每日情绪天气）· AI 时事扣星象 · 自我深化（每日一题越答越准）· 关系网（每段关系一个合盘话题）· 事件专题（水逆/满月/本命年仪式）· 回看（「一年前的今天你的盘说…」）。

## 6. 变量奖赏：天空自带老虎机

有些天「平静无相位」（休息，避免疲劳），有些天「⚡重要相位」（高潮）。这种不可预测起伏由天空免费提供，是变量奖赏，且不靠制造焦虑（守红线）。

## 7. 召回堆叠（H5 现实：没有原生 push，推送 + 被动开启并存）

**通道已定：纯 Web Push + Email**（不上 WeChat 服务号 / Telegram bot）。四层叠加：

1. **⭐ Keystone · PWA 加到主屏（命门）**：一个动作解锁两件事——① 桌面那只眼图标＝被动习惯线索（每天看得到）② Web Push 生效（含 iOS）。**做成首读激活峰后的一等公民步骤，峰值触发**（「把 Iris 加到桌面，每天第一件事先看今天」）。
2. **Web Push（推送主力，但有限）**：安卓浏览器直接可用；iOS 必须先加桌（16.4+）；opt-in 率低 + 受众偏 iPhone → 覆盖是「奖励」非「地基」。
3. **Email（可靠兜底，非每日）**：用于周摘要 + 大相位事件 + 流失召回（「3 天没见，你的盘又准了一点」）。每日推易 spam。**注册提供 Email 选项（首选），确保拿到可送达邮箱喂召回**（Apple 登录有「隐藏邮箱」，故须显式 Email 注册）；登录＝Email + Google + Apple。
4. **⚠ 被动开启层（诚实）**：未加桌/未开 push 的用户，回访靠桌面线索 + 好奇缺口（「今天说什么」）+ 明日钩子。**「今日」体验本身必须够勾，不能指望推送替它拉人**。H5 日活比原生难，是真实代价。

> **北极星护栏指标之一：无 push 下的次日回访率**——它衡量「今日」体验本身的拉力，独立于推送覆盖。

## 8. 深度池

「今日」负责天天来；深度池（四痛域深度、遇事追问、合盘、解锁）负责来了走不掉。
