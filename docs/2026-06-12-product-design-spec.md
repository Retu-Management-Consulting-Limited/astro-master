## 1. 产品本质（第一性定义）

> 一台以**占星象征为介面**、以 **AI 自我建模为引擎**、向用户回渲一个日益精确之自我模型的**意义—连贯性仪器**。它出售被代谢的不确定性。

操盘语言版：**占星是她愿意伸进来的那只手，AI 是越收越紧的那只手。卖的不是运势，是「终于有一个东西比我自己更懂我，而且我离不开它」。**

- 占星因果机制在现有证据下为空集 → 产品交付的不是天象资讯，是「被看见」与不确定性的降低。
- 因此：**占星是皮肤（介面），自我建模是器官（引擎）**。一切设计以「扩大『不确定→连贯』的跃迁、且不依赖可证伪的虚假主张」为单一判准。

## 2. 目标市场（beachhead）

**全球小红书华人女性**（北美/澳洲/欧洲，留学/移民，20s–30s，小红书重度）。
- **产品双语（中 / 英），i18n 内建**：中文服务 beachhead，English 就绪以备国际化（Iris 名即为此选）。Iris 声音须双语原生、非翻译（金句各自原创）。
- 痛域排序（**A 重排，财富升为主打**）：主打三域＝**感情 · 财富/财运 · 孤独身份夹缝**；次＝自我/方向、事业/AI。财富/财运由「财运日历」主打（强留存+高意向）；**孤独/被看穿仍是差异化灵魂**（测测覆盖不到），财运内容须经「懂你」个人化，不沦为商品化今日财运。
- 渠道：小红书为主、IG 次之。审美门槛高（林星审美关）。
- 人物：小漾🚀传播 / 林星⭐信誉 为核心；陈昊💰营收 / 秦姐🛡️留存 为后续。
- 详见 [人物形象](2026-06-12-customer-personas.md)（注：原稿 mainland 设定，待海外 pass）。

## 3. 核心体验

**C 引导式骨架 + B 追问对话**：进场一段「开场速读」钩子 → 主题导览（解空白页、切付费单位）→ 每主题内嵌聊天追问。详见 [IA](2026-06-12-information-architecture.md)。

## 4. 核心引擎与护城河

- **🥇 自我模型引擎（第一护城河 · Phase 1 必交付）**：持续演化的「她是谁」画像 + 跨 session 记忆 + 定期综合，条件化每次解读。**越用越准、三个月后比你闺蜜还懂你**——赢测测的那张牌。关键区分：盘校准（出生时间）两三周封顶＝前期钩子；自我模型无天花板＝真护城河；护城河本质是「关系」不是数据。详见 [自我模型引擎](2026-06-12-self-model-engine.md)。
- **亮点偵測**：先扫她的盘、抓最突出的 2–3 个配置，据此个人化生成主题顺序与追问 chips。「同一 app，每人引导都不同」。
- **潜伏式校准**：把每次互动当证据贝氏收窄出生时间，盘越来越准（前期钩子，两三周封顶）。每个潜伏触点须同时真回馈价值（埋伏≠偷取）。

## 5. 设计原则（收割引擎）

**收割飞轮**：①制造「被看穿」→ ②一刀一糖 → ③背后偷偷变准 → ④删不掉 → ⑤情绪溢出=传播 → 拉新回①。
**六武器**：① 卖被看穿不卖准 ② 戳痛必塞糖 ③ 背后偷偷变准 ④ 暗箱藏死 ⑤ 传播=接住情绪溢出 ⑥ 红线：别把人戳到崩溃（杀鸡取卵）。
**AI 世代定位**：冥王水瓶（2024–2043）＝AI 颠覆的占星签名。「当 AI 让一切不确定，回到你的星盘，找回 AI 拿不走、只属于你的人生坐标。」
详见 [核心设计原则](2026-06-12-core-design-principles.md)。

## 6. 大师人设

- **不叫「大师」，名字 Iris**（虹膜＝眼，名与标志合一；双语原生便于国际化；名暖声毒＝天鹅绒手套里的铁拳）。视觉识别＝一只眼（瞳孔里一片星空），「看进你心里」非「从天上盯你」。
- **不承认是 AI**（沉浸/暗箱藏死），但≠假装真人；角落留极轻 AI 揭露（合规 + 成熟用户信任）。
- **声音：C 毒舌通透为底色**，A 暖/B 洞察/D 神秘为旋钮。
- **自适应语气引擎**（bandit，奖赏＝持久留存 ∩ 接住约束）+ 两道硬覆盖（冷启动用固定默认「锐但已接住」；脆弱安全阀强制 A/B）。
详见 [大师人设](2026-06-12-master-persona.md)。

## 7. 关键流程

- **进场漏斗（激活峰）**：小红书链接 → 落地（先抛痛点）→ 输入**出生日期 + 出生时间（不知道→校准小测）+ 出生地（国家+城市，供 geocoding + 历史时区）**（**输入页不放登录**）→ **校准小测**（埋上升 + 第一刀被看穿）→ **首读** →〔她首次做 gated 动作（追问/存卡/看更多）触发〕**注册留存（先体验后注册**：先给惊艳再要她留下；**收昵称**=显示名；Email 首选为召回 / Google / Apple）→ **今日**。穿插「排盘中 / 解读中」loading 仪式屏（把死等变期待）；**PWA 加桌＝财运日历停留 30s 的峰值 overlay，非固定漏斗步**。首读 = 整条曲线命门，对着「一个具体的她」说话（离散孤独 + 感情），C 锐但已接住，结尾给「存成卡片」。
- **每日仪式（昨/今/明）**：昨=验证（喂模型+坐实准，**验证回溯、问内在体验**）/ 今=指引 / 明=钩子。快慢分层（月亮日常 / 外行星章节）。内容由行运程序化生成，永不枯竭。详见 [每日仪式](2026-06-12-daily-ritual.md)。
  - **早期留存（直打头号死因）**：5 天钩子弧（分阶段揭晓 + Day3 约定肖像 + 可感「越来越准」）撑过头 3-5 天，到 Day5 自我模型/行运接棒。
  - **PWA 无底线诱导**（召回命门）：峰值后闸门式诱导加桌（iOS 只能诱+教，不能强制）；先给糖再要承诺。
- **💰 财运日历**：绿→白→红整月网格（越旺越绿/越慎越红）。旺=财运旺+投资胜率高（轻免责）；慎=别投资别大额（护住她）。日历＝回访机器（强留存）；两条红线：行动时机非财富预测、赋能非贩卖焦虑/赌博。详见 [财运日历](2026-06-12-wealth-calendar.md)。
- **本命/盘面**：高亮驱动（只亮 2–3 个）、互动地图、越用越亮。详见 [盘面美学](2026-06-12-chart-visual-design.md)。

## 8. 增长 / 传播

- **金句卡**（一对多广播）：情绪峰值才弹、不露脸（可选昵称）、CTA 克制靠「求链接」。详见 [金句卡](2026-06-12-viral-share-card.md)。
- **合盘**（强制双人，**K 因子 >1 引擎**）：主推邀请链接（B 自填、B 也走激活峰）、多维分数讲故事、UI 极简、镜不预言 + 分数永不判死刑。详见 [合盘](2026-06-12-synastry-compatibility.md)。

## 9. 平台

**H5 mobile web（PWA 增强）· 海外华人**。零安装链接即传播；Google/Apple 登录；Stripe（USD，免苹果税）；Vercel 全球、免 ICP 备案。**召回＝Email + Web Push + PWA 加桌**（非微信服务号）。首屏性能命脉。详见 [核心设计原则 §5](2026-06-12-core-design-principles.md)。

## 10. 排盘算法

客户端（astronomy-engine, MIT）即时显示 + 服务端解读/自我模型（单一真相源，同库防漂移）。算 10 行星+四角+凯龙/北交/莉莉丝，Placidus，5 主相位，亮点偵測按相位紧密度排序。算得丰富、显得克制。坑：历史时区/DST、geocoding。详见 [排盘算法](2026-06-12-chart-computation.md)。

## 11. 设计不变量 & 自查

验证回溯/预测前瞻；具体性给内在、谦抑给外部；镜不是预言机；戳准接住；出图前对照领域机制（流畅是黄灯）。详见 [核心设计原则 §4](2026-06-12-core-design-principles.md)。

## 12. 明确不做 / 待办

- **变现先不做**（MVP 专注前期人性收割机 + 传播），付费单位 🔒 仅占位。
- **实作方式：完整设计阶段先行**——UI/UX 是竞争力（颜值=想晒=传播燃料），故实作 plan 的 Phase 1 ＝ 完整 hi-fi 设计系统 + 所有关键屏及状态（用 frontend-design），工程在设计定稿后实作。
- **双语定位已定（中/英），不做海外 pass**：旅程图 & 人物画像的 mainland 风味细节（¥、城市、微信）视为示意，统一读作海外/双语/USD/小红书·IG。
- **名字已定 Iris**；logo 已定（A 占星虹膜眼＝主标 / B 极简线条眼＝app 图标 / C 星空虹膜＝启动页）。
- **合盘入口显眼度**、**tab 命名**待定。
