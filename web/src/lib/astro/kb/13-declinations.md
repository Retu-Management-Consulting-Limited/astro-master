# 13 — 赤緯與平行相位 Declinations and Parallels

赤緯是黃經之外的另一個座標系統。**多數現代占星師忽略赤緯，但古典與部分現代學派把它視為重要補充。**

## 1. 赤緯的基本概念

### 兩個天球座標系統

| 系統 | 經度（橫向）| 緯度（縱向）|
|------|-----------|-----------|
| **黃道座標 Ecliptic** | **黃經 Longitude**（占星主要用）| 黃緯 Latitude（行星偏離黃道的角度）|
| **赤道座標 Equatorial** | 赤經 Right Ascension | **赤緯 Declination**（占星偶用）|

**占星上的「相位」通常基於黃經（黃道上的角距離）。** 赤緯是另一條獨立軸線，提供「南北向」的關係。

### 赤緯的範圍

- 太陽的赤緯範圍：**約 -23.45° 到 +23.45°**（對應冬至 / 夏至）
- 月亮赤緯範圍：**約 -28.5° 到 +28.5°**（因月球軌道相對黃道有 5° 傾角）
- 行星赤緯範圍因軌道傾角而異（多數 < 25°）

### Out of Bounds (OOB) 越界

**「越界」 = 行星赤緯超過太陽的最大值（±23.45°）。**

- 月亮：每月可能有幾天 OOB（赤緯超過 ±23.45°）
- 水星 / 金星 / 火星：偶爾 OOB
- 太陽永遠不會 OOB（它定義了邊界）

## 2. Out of Bounds 解讀

### 核心象徵
**OOB 行星跳出「太陽法則」、不受常規約束。**

行星 OOB 的人，該行星象徵的能量在生命中表現為「**極端、突破常規、失序、天才、不安定**」。

### 各行星 OOB 的詮釋

#### Moon OOB（最常見）
- **約占人口 10–15%**
- 童年沒被「常規養大」、家庭結構非標準（離婚、流動、過早成熟、被迫獨立）
- 情感 / 直覺發展不依常規
- 常見於：藝術家、創新者、政治反叛者
- 著名 OOB Moon 例：Hillary Clinton, Steven Spielberg, Brad Pitt 等

#### Mercury OOB
- 思維跳脫常規邏輯、語言創新、可能話太多或太奇怪
- 可能「神童」型 / 也可能溝通失序
- 常見於：發明家、瘋狂科學家、天才作家、思考方式不主流

#### Venus OOB
- 對愛 / 美 / 價值的標準與社會脫節
- 可能極端羅曼蒂克或反傳統的關係模式
- 多元戀 / 非主流性向 / 跨文化關係常見

#### Mars OOB
- 行動 / 慾望的力度極端、難以受社會規範約束
- 可能領袖也可能挑戰權威者；運動員 / 戰士
- 常見於：極限運動員、革命者、軍人英雄

#### Jupiter OOB（罕見）
- 信念 / 視野跳脫常規 — 哲學突破或極端教條
- 國際性影響的命主常見

#### Saturn OOB（極罕見）
- 結構觀念極端 — 可能完全反建制或建立新體系
- 非常稀有，當代天文觀察會持續變動

> **OOB 不必然好或壞** — 是「能量超出常規軌道」。當事人需要學習「如何馴服這股力量」 — 否則容易成為毀滅性。

### OOB 的時間性
- **本命 OOB：** 一生帶有該主題
- **行運 OOB：** 短期內（OOB 期間）該行星啟動的事件帶有極端 / 突破性

## 3. Parallel / Contraparallel 平行 / 反平行

赤緯的「相位」 — 兩顆行星的赤緯關係：

### Parallel（平行）
- **兩顆行星赤緯相同方向、值接近**（容許度通常 1°）
- 例：A 行星赤緯 +12°15′，B 行星赤緯 +12°30′ → parallel
- **詮釋：** 類似 conjunction — 兩股能量「平行同步運作」、加強合作
- **強度：** 比 conjunction 略弱，但同向加強

### Contraparallel（反平行）
- **兩顆行星赤緯方向相反、絕對值接近**（容許度 1°）
- 例：A 行星赤緯 +18°，B 行星赤緯 -18° → contraparallel
- **詮釋：** 類似 opposition — 「南北向兩極對話」
- **強度：** 比 opposition 略弱

### 為什麼重要

#### 補強或糾正主盤判讀
- 兩行星在黃經上**沒有相位**（不在容許度內）但**有 parallel** → 仍有「隱性連結」
- 例：Sun 與 Mars 在黃經上 30° 相距（半六分，弱）但 contraparallel → 兩星實有強烈的「對立性張力」，不應僅按半六分弱相位處理

#### 加強現有相位
- 黃經上有相位 + 赤緯也有 parallel/contraparallel → 該相位更強
- 例：Venus square Saturn（90°，黃經）+ parallel（赤緯）→ 結構性愛之限制議題加倍強烈

### 計算

赤緯數據在多數占星軟體（Solar Fire, Astro Gold, Astro.com）都有顯示。AstroSeek 等網站也提供 declination 報表。

## 4. 平行相位的詮釋實例

### Sun parallel Moon
- 太陽月亮在赤緯上同步 → 自我意識與情感運作和諧（補強或代替黃經合相）
- 命主內在「想要的」與「需要的」較一致

### Sun contraparallel Saturn
- 自我與結構責任在「南北向」對立 → 內在父權批判 / 自我實現的張力
- 補強或加倍黃經 Sun 對沖 Saturn 的議題

### Mars parallel Venus
- 行動與愛 / 美感的同步 — 強烈魅力 / 性吸引力或創造性激情
- 不同於黃經 Mars-Venus 相位，這個是「赤緯共振」 — 命主可能透過行動展現美感（運動藝術家、激情舞者）

### Moon parallel Pluto
- 情感與深度轉化的同步 — 強烈情感深度 / 容易進入心理深淵
- 補強月亮 Pluto 主題

## 5. 赤緯與本命解盤

### 整合步驟

#### Step 1：列出本命 OOB 行星
- 哪些行星超過 ±23.45°？
- 月亮 OOB 是最常見且影響最深刻

#### Step 2：列出 parallel / contraparallel 配對
- 通常軟體會自動標出
- 容許度 1° 內為強，1°–2° 為中

#### Step 3：整合到主解讀
- OOB 行星 → 該主題的「極端 / 突破」傾向
- Parallel = 加強同步主題
- Contraparallel = 加強對立主題

#### Step 4：與主盤交叉印證
- 黃經有相位 + 赤緯加強 → 高優先主題
- 黃經無相位 + 赤緯有相位 → 隱性主題，可能比表面看到更強

## 6. 行運中的赤緯

### Eclipses 食的赤緯機制
- **食 = 太陽 / 月亮在月交點附近精確同步合相 / 對沖。** 這也意味著它們的**赤緯接近** — 因為月交點是黃道與月軌的交點，月亮在交點附近赤緯接近黃道（即太陽附近）。
- 強食點通常是「赤緯也對齊」的點。

### 行運 OOB 行星
- 當行運 Mars / Venus / Mercury 處於 OOB 期間 → 該行星行運啟動的事件帶有極端 / 突破性
- 例：行運 Venus OOB 期間，可能發生不尋常的關係事件、突破性的愛情、或關於價值觀的極端決定

## 7. 派別差異

| 學派 | 對赤緯的立場 |
|------|-----------|
| 古典 / 希臘化 | 部分使用 — 古代天文學中赤緯（OAK = Oblique Ascension Kingdom 等概念）有用 |
| 中世紀 | Antiscia 系統（赤緯的對映點，類似 declination concept）|
| 現代心理 | 多忽略；少數心理派（如 Tracy Marks）採用 |
| Cosmobiology | 部分採用，特別是 parallel 加強硬相位的判讀 |
| Bernadette Brady 學派 | 重視 parallel + 全面 fixed star paran 系統 |
| 現代 OOB 研究 | 由 Kt Boehrer 等推廣，OOB 研究成為當代「補充流派」 |

> **本 skill 預設：** 解盤主視角不深入赤緯，但**月亮 OOB 永遠檢查**（影響太大）。其他行星 OOB 與 parallel 在以下情境啟動：
> - 主盤詮釋遇到難解主題（例：「為什麼這個人這麼極端」）→ 檢查 OOB
> - 主盤相位網絡稀疏 → 檢查 parallel 補強
> - 命主出生資料有極端配置（如行星 OOB 集中）→ 必須提及

## 8. 重要的「Antiscia 反射點」 (中世紀概念)

**Antiscia：** 黃道 0° Cancer / 0° Capricorn 軸線（夏至 / 冬至點）的反射對應。
- 例：10° Aries 的 antiscion = 20° Virgo（反射過 0° Cancer 軸）
- 中世紀占星師視 antiscia 為「隱性合相」 — 當行星 / 軸線在彼此 antiscia 上時，有隱性連結

**Contra-antiscia：** 0° Aries / 0° Libra 軸線（春分 / 秋分點）的反射。

> 這是赤緯邏輯在黃道上的對應（夏冬至點是赤緯的極值），詮釋上類似 parallel — 隱性同步主題。
> Antiscia 在現代多被忽略，但古典派（特別是中世紀復興派）仍在使用。
