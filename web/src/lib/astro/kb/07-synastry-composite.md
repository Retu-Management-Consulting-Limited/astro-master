# 07 — 關係占星 Synastry & Composite

關係占星處理「兩個人 / 多個人之間的能量動態」。本章涵蓋四種核心技術：比對盤 (Synastry)、組合中點盤 (Composite)、Davison 盤、關係 transits。

## 0. 核心觀念

1. **關係占星看的是「動力」而非「適配度」。** 沒有絕對的好 / 壞 synastry，每段關係都有它的功課與禮物。
2. **多重技術交叉印證才可靠。** 單獨看一張比對盤判定關係好壞是過度簡化；合用 synastry + composite + transits 才能看出全貌。
3. **不做命定預言。** 「你們合不合」是占星師最被誤用的問題；應該回答「這段關係的核心動力是什麼、需要學什麼」。
4. **關係不止是浪漫關係。** 親子、朋友、商業夥伴、敵人都能用 synastry 分析。

## 1. Synastry 比對盤

**做法：** 把 A 的命盤行星疊加到 B 的命盤上（或反之），看 A 的行星落在 B 的哪些宮位、與 B 的哪些行星形成相位。

**雙向都要看：** 「A 的 Mars 對 B 的 Venus」與「B 的 Mars 對 A 的 Venus」是不同的動力。

### 解讀步驟

#### Step 1：互相落宮 (Houses)
A 的行星落在 B 的哪些宮位 → A 在 B 的生活中啟動哪些領域。

| A 的行星落在 B 的… | 關係動力 |
|------------------|---------|
| 1 宮 | A 對 B 是「鏡子 / 自我延伸」，互相塑形 |
| 4 宮 | A 喚起 B 的家 / 安全感 / 童年議題 |
| 5 宮 | 戀愛、玩、創意、親子（5 宮也代表孩子）|
| 7 宮 | 一對一伴侶、合夥、公開敵人（依關係性質）|
| 8 宮 | 深度親密、性、共享資源、心理整合 |
| 10 宮 | 影響 B 的事業、社會角色 |
| 12 宮 | 隱藏的、潛意識的、業力性的連結 |

#### Step 2：行星間相位 (Inter-Aspects)

##### 主要組合與其動力

| 行星組合 | 核心動力 |
|---------|---------|
| **Sun - Moon**（一方陽光、一方情感容器）| 經典「陰陽」配，互補感強 |
| **Sun - Sun** | 自我意識的對話；同星座共鳴、衝突看相位 |
| **Moon - Moon** | 情緒共振；同元素 → 自然舒服 |
| **Venus - Mars** | 性吸引 / 浪漫吸引的核心；和諧相位 = 自然 chemistry |
| **Venus - Venus** | 美感 / 價值觀對話 |
| **Mars - Mars** | 行動風格與衝突風格；困難相位 = 經常爭執 |
| **Sun - Saturn** | 結構與權威動力；困難相位 = 一方感到壓抑、另一方感到沉重 |
| **Moon - Saturn** | 情感被壓抑 / 被結構化；可極穩定也可極壓抑 |
| **Venus - Saturn** | 「責任之愛」、commitment 議題；困難 = 愛中的限制感 |
| **Mars - Saturn** | 行動 vs 限制的拉扯，常見性慾 / 競爭壓力 |
| **Sun - Pluto** | 強烈吸引、權力動力；極化關係 |
| **Moon - Pluto** | 情感深度與控制議題、命定感 |
| **Venus - Pluto** | 命中註定的愛、佔有與失去的張力 |
| **Mars - Pluto** | 性與權力的最強相位、可極致也可破壞 |
| **Sun - Neptune** | 理想化、迷霧、靈性連結 |
| **Venus - Neptune** | 浪漫迷醉、美感共鳴、易見不到對方真實 |
| **Sun - Uranus** | 突發吸引、自由議題、不可預測 |
| **Moon - Neptune** | 情緒共感、易模糊邊界 |

##### 重要相位的「方向性」
A 的 Saturn 合 B 的 Sun → A 是「結構提供者」，B 是「被結構化者」。互動上 A 容易扮演「老師 / 父母 / 限制者」，B 容易感受到 A 的權威或壓力。

#### Step 3：軸線觸碰 (Angles Contacts)
- A 的行星合 B 的 ASC / DSC / MC / IC 是極強連結
- 合 ASC → A 對 B「整體影響強」、外顯如同「另一個自己」
- 合 DSC → A 是 B 投射伴侶原型的對象
- 合 MC → A 影響 B 的事業 / 社會角色
- 合 IC → A 觸動 B 的家 / 根 / 心理深處

#### Step 4：交點 (Nodes) 連結
- A 的行星合 B 的 NN → A 推動 B 往演化方向（命定感的連結）
- A 的行星合 B 的 SN → A 喚起 B 過去模式（業力感的連結，也可能阻礙演化）
- 雙方 NN/SN 軸互相連結（180° 對軸） → 強烈業力 / 命運連結

#### Step 5：Vertex 觸碰
- A 的行星 / 軸線合 B 的 Vertex → 命定相遇感
- 雙方 Vertex 互相連結 → 共同命運線索

#### Step 6：Chiron 連結
- 雙方 Chiron 相位 → 互相觸動傷口、有療癒可能也有重複傷害的風險
- A 的 Chiron 合 B 的 Sun / Venus / Moon → A 觸碰 B 內在傷口

### 解讀整合方法

#### 動力分類
把 synastry 中的相位分為三類：

1. **吸引動力 (Attraction Dynamics)：** Venus-Mars、Sun-Moon、Sun-Pluto 等強烈互動
2. **舒適動力 (Comfort Dynamics)：** 同元素三軸、Moon-Moon trine、Venus-Venus 和諧
3. **挑戰動力 (Challenge Dynamics)：** Saturn / Pluto / Mars 的困難相位、四軸對沖

**只有 1 沒有 2 → 強烈吸引但難舒適**
**只有 2 沒有 1 → 舒適如朋友但缺火花**
**只有 3 → 不斷衝突但無連結基礎**
**1+2+3 都有 → 有深度發展可能性的關係**

#### 重複主題 (Recurring Themes)
找出在 synastry 中被多次點出的主題（例：多個 Saturn-Venus / Saturn-Moon / Saturn-Sun → 結構責任議題是這段關係主軸）。

## 2. Composite Chart 組合中點盤

**做法：** 把 A 與 B 各自的行星位置取「中點 (midpoint)」，組成一張新的盤。例：A 的 Sun 在牡羊 10°，B 的 Sun 在天秤 20° → composite Sun 在巨蟹 0°（或摩羯 0°，取較近的）。

### 與 Synastry 的差異
- **Synastry：** 看 A 與 B 之間的「互動動力」
- **Composite：** 看 A 與 B 「合在一起變成的『第三個實體』」 — 關係本身的命盤

### 解讀重點

#### Composite 重點觀察項
1. **Composite Sun**：關係的核心目的、為何而存在
2. **Composite Moon**：關係的情感基調、安全感模式
3. **Composite Ascendant**：關係給外界的形象、關係的入口
4. **Composite 7th house**：關係的對外面、互動模式
5. **Composite Venus**：關係中愛 / 美的展現
6. **Composite Mars**：關係的衝突與行動風格
7. **Composite Saturn**：關係的結構責任、長期性
8. **Composite Pluto**：關係的轉化深度、權力動力
9. **Composite NN**：這段關係的演化方向

#### Composite 的特殊提醒
- Composite 盤沒有「出生時間」，所以軸線（ASC, MC）的計算方式有兩種：
  - **Reference location 法：** 以雙方目前所在地（或預期共同生活的地方）為基準
  - **Equal house from composite Sun：** 把 composite Sun 放 1 宮，依次排其他宮（較少用）
- Composite 軸線意義可能因方法選擇而不同，**解讀時要明示用什麼方法**。

#### Composite 的相位
- 看 composite 行星之間的相位 = 關係內在動力
- composite 相位緊密（< 3°）= 顯化強烈
- composite 行星與雙方各自命盤的相位也有意義（進階分析）

### Composite 的限制
- Composite 盤是「虛擬實體」，沒有實際出生時間 → 對行運敏感度爭議。
- 主要用於性格化關係本身，**不適合用來算流年**。

## 3. Davison Chart 戴維森盤

**做法：** 取 A 與 B 出生時間的「中點」（時間 + 經緯度都取中點），計算這個「中間時點 / 中間地點」的命盤。

### 與 Composite 的差異
- **Composite：** 行星位置的中點（虛擬計算）
- **Davison：** 真實的時空中點 → 是一張「實際存在」的盤
- **意義差異：** Composite 偏向「兩人的內在融合」、Davison 偏向「兩人的外在共同表現」

### 何時用 Davison
- 想看關係的「外顯時間性」（例：關係的開始時點、共同事件時機）
- Composite 結果不清楚時補充
- 想看 transits 對關係的影響時，**Davison 因為是真實時點所以可以受 transits 影響**，這是 composite 做不到的優勢

### 注意
- Davison 的中點時間 / 地點是數學計算結果，不必然「有意義」（例：兩人時差 30 年 → 中點時間在沒有任何事發生的時刻）。
- 多數現代占星師在 composite 與 Davison 中選擇 composite 為主，Davison 為輔。

## 4. Relationship Transits 關係行運

關係不是靜態的，會隨時間發展。看「行運如何啟動 synastry / composite 的關鍵點」是判斷關係階段的核心方法。

### 啟動 Synastry 的行運
- 行運觸碰雙方命盤 synastry 中的「熱點」（例：當前 Saturn 行運合 A 的 Venus（這個 Venus 也是 B 的合相點） → 這段關係的責任議題正被啟動。

### 啟動 Composite 的行運
- 行運合 / 衝 composite Sun → 關係核心目的的危機 / 確認時刻
- 行運合 composite Saturn → 關係的結構性測試
- 行運合 composite Pluto → 關係的深度轉化
- 行運合 composite NN → 關係朝演化方向推進

### 重要轉折點
- **第一次 Saturn 半回歸（關係開始後 7-7.5 年）：** 古典「七年之癢」對應的占星時點 — 關係結構面臨第一次重大檢視
- **Composite Saturn 的行運觸發：** 通常對應關係的承諾 / 結束點
- **食點觸發 composite 軸線：** 重大事件 / 公開化 / 結束

## 5. 同性 / 多元關係的占星詮釋

傳統占星文獻常假設異性戀框架，但占星本身的象徵語言並無此預設。

### 詮釋原則
- **不必把 Sun 強配為男性、Moon 為女性。** 兩顆行星都在每個人盤裡。
- **Venus / Mars 軸（陰柔 / 陽剛能量）也不必對應生理性別。** 一個人的 Mars 強的伴侶，無論性別，都是「驅動 / 主動」的動力源。
- **同性關係常有 Sun-Sun、Moon-Moon、Venus-Venus 的同類連結放大現象。** 這不是「缺少互補」，而是「同質的深度相處」。
- **跨性別 / 非二元命主的盤解讀，** 重點在命主自己對 Sun / Moon / Venus / Mars 各能量的關係，而非生理性別預設。

## 6. 關係解讀的 5 步輸出結構

當給命主呈現 synastry / composite 解讀時：

```
1. 整體判定 (Overview)
   - 這是怎樣的一段關係？（命運感、舒適感、挑戰感的比重）
   - 主要動力的「形狀」

2. 吸引機制 (What Draws You Together)
   - Venus-Mars、Sun-Moon 等吸引性相位
   - 為何「想要彼此」

3. 共處模式 (Daily Dynamics)
   - 落宮分析
   - 雙方在彼此生活中啟動哪些領域
   - Composite Sun / Moon / ASC 整體基調

4. 核心議題 (Core Themes)
   - 從反覆出現的相位主題抽出 2-4 個關係課題
   - 例：「自由 vs 安全」「成長 vs 穩定」

5. 演化方向 (Evolution)
   - 雙方 NN/SN 連結 → 業力 / 命定主題
   - Composite NN → 關係要往哪去
   - 整合敘事
```

## 7. 派別差異總結

| 議題 | Hellenistic | Modern Psychological | Evolutionary |
|------|-------------|---------------------|--------------|
| 主要技術 | Synastry by sign + Lots（婚姻點等） | Synastry inter-aspects + Composite | Synastry node 連結 + Pluto 連結 |
| 婚姻適配判定 | 用婚姻點、Venus/Mars 狀態、Lord of 7th 古典規則 | 不做適配判定，看心理動力 | 看靈魂連結與業力主題 |
| Composite | 不使用（古典系統無此概念） | 主要工具 | 主要工具，但更看 nodal 連結 |
| Karma 業力 | 較少強調 | 看 Saturn / Chiron 連結 | 核心概念，看南交點 / Pluto / Saturn 連結 |
