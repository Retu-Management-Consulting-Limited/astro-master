# 04 — 相位 Aspects

## 0. 相位的本質

**相位是兩顆行星在黃道上形成的角度關係。** 它揭示這兩股能量在一個人 / 一段時間內如何互動 — 相助、衝突、整合、迴避。

**沒有相位的行星 (unaspected planet)** 在盤中相對「孤立運作」，其象徵領域常與其他生命主題切離（既有獨立性，也容易表現極端或社會適應失調）。

## 2. 主相位 (Major / Ptolemaic Aspects)

托勒密 (Ptolemy) 系統的五個主要相位，是占星詮釋的骨幹：

| 相位 | 角度 | 符號 | 性質 | 能量質地 |
|------|------|------|------|---------|
| 合相 Conjunction | 0° | ☌ | 中性（依行星 / 星座定吉凶）| 融合、強化、新循環起點 |
| 對分 Opposition | 180° | ☍ | 困難 | 對立、投射、需要整合兩極 |
| 三分 Trine | 120° | △ | 和諧 | 流動、天賦、習以為常 |
| 四分 Square | 90° | □ | 困難 | 緊張、行動驅力、危機推動 |
| 六分 Sextile | 60° | ⚹ | 和諧 | 機會、需主動使用、社交 |

### 各主相位深入

#### ☌ Conjunction 合相（0°）
- 兩顆行星融合為一個動力單位，呈現「複合性質」。
- **好壞依行星本質而定：** Venus + Jupiter = 大幸運合相；Mars + Saturn = 古典凶煞。
- 若兩行星元素相同（同星座）→ 元素一致的合相，能量純粹但缺對話。
- 若跨星座（out-of-sign conjunction）→ 度數合但風格不一致，有張力但能整合（古典派強調 in-sign 才算真合）。
- 與太陽的合相要看距離：8.5° 內 = combust（焦傷）；17 角分內 = cazimi（核心強化）。

#### ☍ Opposition 對分（180°）
- 兩顆行星分別在對宮 / 對立星座 (例：牡羊 vs 天秤；2 宮 vs 8 宮)。
- 主題是「鏡像對立、需要整合的兩極」。
- 對分常透過「他者」展現 — 把一極投射到伴侶 / 對手身上，看見後再收回。
- 同元素不同模式（如牡羊火 vs 天秤風）— 兩極之間實有橋樑可建。
- **整合難度：** 中高。整合後的成果常是命主的「慧見來源」。

#### △ Trine 三分（120°）
- 同元素的星座之間（如牡羊 / 獅子 / 射手都火）。
- 兩股能量自然流動、無摩擦、對命主而言是「天賦」。
- **陰影：** 太順=不被察覺=不珍惜=惰性。三分需要被「困難相位」激活才能發揮。
- **大三角 (Grand Trine)** 是三顆行星形成 120-120-120 的閉合三角，常被誤解為純好運；實際上若整盤無對分 / 四分激活，反而易陷入舒適與停滯。

#### □ Square 四分（90°）
- 同模式的星座之間（如牡羊 / 巨蟹都開創；金牛 / 獅子都固定）。
- 兩股能量被「同樣的衝動 (模式)」驅動，但展現方式 (元素) 不同 → 內在結構性衝突。
- **行動驅力的主要來源。** 沒有四分的盤通常缺乏「危機推動」的張力。
- **整合方法：** 不能消除四分，要學會在兩股能量之間切換而非二選一。

#### ⚹ Sextile 六分（60°）
- 互相相容元素之間（火 - 風、土 - 水）。
- 不像三分那樣「自動發揮」，需要命主**主動行動**才能啟動。
- 是「機會 + 努力」相位 — 若不行動，潛力閒置。
- 常與第三方介入相關（透過他人連結而成事）。

## 2. 次相位 (Minor Aspects)

| 相位 | 角度 | 符號 | 性質 | 詮釋 |
|------|------|------|------|------|
| 半六分 Semi-Sextile | 30° | ⚺ | 弱張力 | 相鄰星座，不適應、需要調整 |
| 半四分 Semi-Square | 45° | ∠ | 小困難 | 內在摩擦、煩躁 |
| 補十二 Quincunx (Inconjunct) | 150° | ⚻ | 中度困難 | 不對稱、無共通點，需轉換才能整合 |
| 一又半 Sesquiquadrate (Sesquisquare) | 135° | ⚼ | 小困難 | 與半四分同類，外顯張力 |
| 十二分 Quintile | 72° | Q | 創造性 | 才華、創新模式（克卜勒系列） |
| 雙十二分 Biquintile | 144° | bQ | 創造性 | 才華、深度創意 |
| 七分 Septile | 51.43° | s | 命運感 | 神秘 / 命運性連結（罕用） |

### 重要次相位深入

#### ⚻ Quincunx / Inconjunct 補十二（150°）
- 兩星位於彼此 6 宮 / 8 宮位置 — 「不見」(no major aspect by sign)。
- 兩種能量無共通元素、無共通模式。
- **詮釋：** 兩股能量「無法對話」，只能轉化、放下、犧牲一邊。常與健康議題（6 宮）或危機轉化（8 宮）對應。
- 兩個補十二指向同一行星 → **Yod（手指）圖形**，見下節。

#### Q Quintile / bQ Biquintile 五分系列
- 克卜勒 (Kepler) 強調的「天才相位」。
- 與獨特創造模式、藝術才華、能在他人看不見處發現結構的能力相關。
- 常見於藝術家、發明家、神秘主義者命盤。

## 3. 容許度 (Orb)

**容許度 = 相位「成立」所允許的偏差度數。**

### 不同學派的容許度標準

#### 古典 / 希臘化（依行星本身的「光線」(Moiety)）
古典占星認為每顆行星都有自己的「光線範圍」，相位容許度 = 兩行星光線的平均：
| 行星 | 光線 (Lilly) |
|------|------------|
| Sun | 15° |
| Moon | 12° |
| Mercury, Venus, Mars | 7–8° |
| Jupiter, Saturn | 9° |
> 例：Sun + Saturn 合相，容許度 = (15+9)/2 = 12°。

#### 現代主流標準
| 相位 | 主流容許度 (本命盤) | 推運 / 行運容許度 |
|------|-------------------|------------------|
| Conjunction, Opposition | 8°（緊：5°）| 1°–2° |
| Square, Trine | 7°（緊：5°）| 1° |
| Sextile | 5°（緊：3°）| 1° |
| Quincunx, Sesquiquadrate | 3° | 0.5° |
| Semi-Sextile, Semi-Square | 2° | 0.3° |
| Quintile family | 1.5°–2° | 0.3° |

涉及 Sun 或 Moon 時，容許度可放寬 1–2°。

### 行星間相位的「精準度梯度」
- **0–1°（exact）：** 命運性、極強烈
- **1–3°（tight）：** 顯著主題
- **3–5°（active）：** 明顯但不主導
- **5°+：** 較隱性，可能要等推運或重大行運啟動才顯化

## 4. 入相位 vs 出相位 (Applying vs Separating)

**入相位 (Applying)：** 較快的行星向慢的行星接近（角度差距縮小） → 能量正在累積 → 事件將要發生。

**出相位 (Separating)：** 較快的行星已離開精確相位（角度差距擴大）→ 事件已過 → 能量正在消化。

### 為什麼重要
- **本命盤：** 入相位 = 性格中正在發展的議題；出相位 = 已內化的成熟模式。
- **卜卦盤 / 擇時盤：** 入相位 = 事情會發生；出相位 = 已結束 / 不會發生。**這是古典卜卦判斷的核心邏輯。**
- **行運：** 入相位行運 = 影響將至；精確時 = 高峰；出相位 = 影響消退。

### 行星速度排序（誰追誰）
慢 → 快：Pluto → Neptune → Uranus → Saturn → Jupiter → Mars → Sun → Venus → Mercury → Moon

> 例：Mars 25° 牡羊與 Saturn 28° 牡羊。Mars 速度比 Saturn 快，Mars 在後追前 → applying conjunction，距離 3°。

## 5. 相位圖形 (Aspect Patterns)

多顆行星的相位組合形成的圖形，是命盤的「結構性簽名」。

### Grand Trine 大三角（三 trines 閉合）
- 三顆行星在同元素，互成 120°。
- **詮釋：** 該元素的能量自然流暢，常為天賦 / 才華領域。
- **陰影：** 太順 = 易耽於舒適 = 不珍惜 = 惰性；常需被外部刺激（行運 / 危機）才能轉化為實際成就。
- 火元素 = 直覺天賦；土 = 物質才能；風 = 思考社交才華；水 = 情感共感天賦。

### Grand Cross 大十字（兩組對分 + 四組四分）
- 四顆行星在同模式，互成 90° / 180°。
- **詮釋：** 強烈的內在張力與行動驅力，常感「四面被擠壓」。整合後是極大成就的動能。
- **陰影：** 易陷在多個衝突中無法突破，需學會接受「無法完全解決」。
- 開創大十字 = 不斷啟動但難收尾；固定 = 固執深層議題；變動 = 焦慮不定。

### T-Square T 三角（一個對分 + 兩個四分指向同一行星）
- **核心原理：** 對分的兩極衝突，在「焦點行星 (apex)」處被引爆。
- **焦點行星 = 命主行動的著力點，也是最大的張力釋放口。**
- 開創 T = 啟動衝突；固定 T = 深層執著；變動 T = 焦慮 / 適應壓力。
- 整合方法：發展焦點行星「對宮」的空缺領域作為平衡（空缺位 = 缺一顆形成大十字的位置）。

### Yod 手指（兩個 quincunx + 一個 sextile，呈 Y 形）
- **形狀：** 兩顆行星 sextile 60°，分別與第三顆 (apex) 形成 quincunx 150°。
- **詮釋：** 「上帝之指」、命運性的 — apex 行星象徵的議題是命主反覆遇到、無法輕易解決的核心命題，常需透過「犧牲、轉換、接受不對等」整合。
- 透過運動學上的「焦點」原理，apex 是命運推動命主必須學的主題。

### Kite 風箏（大三角 + 一顆對分大三角的某點）
- **核心：** 大三角的天賦 + 對分提供的「焦點與動能」。
- 比純大三角更易實現天賦，因為對分迫使命主把流動能量導向具體成就。

### Mystic Rectangle 神秘矩形（兩 oppositions + 兩 trines + 兩 sextiles）
- 四顆行星形成矩形，含兩組對分、兩組三分、兩組六分。
- **詮釋：** 結構平衡、危機與機會交織、能在挑戰中創造和諧。

### Stellium 星群（三顆以上行星集中在同一星座或同一宮）
- 不是嚴格相位圖形，但實質上構成命盤主題的中心。
- **詮釋：** 該星座 / 宮位的議題壓倒性主導命主人生；該領域是「天賦也是課題」。
- 含 Sun / Moon 的 stellium 影響更深。

## 6. 相位的「光輝」(Aspect by Sign)

古典占星還用「按星座」(by sign) 計算相位 — 即只看星座位置不看度數：
- 牡羊與獅子（同火、120° 距離）— **trine by sign**，無論度數差距，都被視為有相位的「對話」。
- 雙子與處女（六宮位 / 150° 距離）— **inconjunct by sign**，視為「不見」。

### 為什麼古典強調 by sign
- 古希臘 / 羅馬看相位的本質是「兩顆行星能否在天上彼此看見 (aspecting)」 — 這是星座之間的「視線」關係，先於度數計算。
- **By sign 解讀方式：** 對於整體主題判斷與 time-lord 技術很重要。
- **By degree (現代 orb 容許)：** 對於精確時機與細節重要。

> **本 skill 詮釋預設：** 兩種都看。先用 by sign 看大結構，再用 by degree 看精確度與容許。

## 7. 相位的方向性與「光線交換」

- **吉星發出吉相位 (sextile/trine)** → 加助
- **吉星發出凶相位 (square/opposition)** → 仍可能助益但帶複雜性
- **凶星發出吉相位** → 古典認為有助力但帶該凶星的特質（Saturn trine 帶來「結構性的禮物」）
- **凶星發出凶相位** → 困難主題，需學習與整合
- **互容 (Mutual Reception)：** 兩顆行星位於對方守護的星座（例：Mars in Cancer + Moon in Aries）→ 兩者「交換家」、產生隱性互助，**即使無相位也能彼此影響**。互容是古典占星非常重要的特殊條件。

## 8. 相位的詮釋方法論

### Step 1 — 標記相位類型
- 主相位 + 容許度內的次相位
- 區分緊（< 3°）vs 鬆（3–8°）

### Step 2 — 區分入 / 出相位
- 對命盤先看是 applying 還是 separating
- 對行運必看 applying，能預判未來高峰時點

### Step 3 — 判定相位質地
- 行星本質（吉 / 凶 / 中性）
- 相位類型（吉 / 凶）
- 兩星宮位 / 星座狀態

### Step 4 — 詮釋整合公式
「[行星 A] (動力) 與 [行星 B] (動力) 透過 [相位類型] 交織 — [相位質地：流暢/張力/機會/衝突] — 在 [宮位場域] 展現。」

範例：
- Sun (核心意志) Square (90°，行動危機) Saturn (結構限制)，Sun in 5th, Saturn in 8th
- → 「自我表達 (5 宮 Sun) 與責任 / 共享資源限制 (8 宮 Saturn) 之間的內在張力 — 創作慾望常被『現實成本』壓住，但這個張力推動命主學習如何把創作落實成能負責的長期作品。」

### Step 5 — 對命主提問（解盤對話）
針對重要相位提開放性問題，例如：
- 「Mars 與 Pluto 的對分，常見於『內在巨大能量 vs 自我控制』的議題 — 你會如何描述自己面對權力或失控的感覺？」

## 9. 派別差異總結

| 議題 | Hellenistic | Modern Psychological | Evolutionary |
|------|-------------|---------------------|--------------|
| 主要使用相位 | 5 個主相位 + by sign | 5 主 + 重要次（quincunx, semisquare） | 5 主 + 對 nodal 軸的所有相位 |
| 容許度 | 行星光線平均 | 標準化的 orb table | 較窄，重視精確 |
| 凶相位處理 | 客觀凶，用以判定不順 | 心理整合的功課 | 過去業力浮現要面對 |
| 大三角 | 助力 | 天賦但易停滯 | 累世帶來的天賦 |
| Yod / Quincunx | 較少強調 | 命運性議題 | 必須整合的演化任務 |
