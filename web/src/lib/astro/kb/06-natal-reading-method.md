# 06 — 本命盤解讀方法論 Natal Chart Reading Methodology

本章是把前五章拼成「一張盤怎麼讀」的工作流程。

## 0. 核心原則

1. **由宏到微。** 先看整體骨架，再進入細節。先說全景再說個別行星。
2. **由結構到內容。** 結構（元素 / 模式 / 半球分布）→ 三軸（日月升）→ 守護鏈 → 相位主題 → 敏感點 → 推運啟動。
3. **不窮舉，要選擇。** 一張盤可以講出五百頁的東西；好的解盤是選出對命主**現在最有意義的 5–8 個主題**。
4. **避免占卜式語言。** 不說「你會 / 你必定 / 你逃不過」；說「傾向 / 容易 / 議題 / 機會 / 課題」。
5. **行星 = 動詞，星座 = 形容詞，宮位 = 場域，相位 = 連結方式。** 永遠回到這個解讀骨架。
6. **同一象徵層次相互印證 → 才是可信主題。** 例：上升射手 + 9 宮 stellium + Jupiter 守護星突出 + Mars 在射手 → 「擴張、追尋意義、出走」是命主極清晰的主題。單一指標不算數。

## 1. 解盤前的準備

### Step 0.1 — 確認資料完整性

| 必要資訊 | 影響項目 | 缺失時的後果 |
|---------|---------|------------|
| 出生日期 | 全盤 | 無法解 |
| 出生時間 | 上升、宮位、月亮（高速時）、四軸 | 只能用日盤 (solar chart) 或時辰矯正 (rectification) |
| 出生地 | 上升、宮位、四軸 | 無法精算宮位與軸線 |
| 出生時區（含當時夏令時） | 行星精確度 | 行星位置可能差數度 |

### Step 0.2 — 確認解盤目的

- **自我探索 / 性格了解** → 偏現代心理派路線
- **預測流年 / 何時做某事** → 推運 + 古典 time-lord（見 `08`）
- **了解人生使命 / 靈魂課題** → 偏演化派路線
- **判斷某具體事件** → 卜卦盤而非本命盤
- **學術理解** → 中性陳述各派觀點

### Step 0.3 — 派別選擇

對應使用者目的選擇主視角：
- 性格議題 → Modern Psychological（Liz Greene、Sasportas）
- 預測 / 時機 → Hellenistic（Chris Brennan、Demetra George）
- 靈魂方向 → Evolutionary（Jeffrey Wolf Green、Steven Forrest）

## 2. 七步解盤流程 (The 7-Step Natal Reading)

### Step 1：判定派系 (Sect)

**Diurnal（日盤）：** 出生時太陽在地平線上（7 宮 - 12 宮）。
**Nocturnal（夜盤）：** 出生時太陽在地平線下（1 宮 - 6 宮）。

| 派系 | Sect light（光體）| 派內吉星 (Benefic of Sect) | 派外吉星 (Out of Sect Benefic) | 派內凶星 (Malefic of Sect) | 派外凶星 (Out of Sect Malefic) |
|------|------------------|---------------------------|------------------------------|--------------------------|------------------------------|
| Diurnal 日 | Sun | Jupiter | Venus | Saturn | Mars |
| Nocturnal 夜 | Moon | Venus | Jupiter | Mars | Saturn |

**派系判定的影響：**
- **派內吉星：** 比平常更吉、更能展現好的一面。
- **派外吉星：** 仍吉但較中性，需特別努力才發揮。
- **派內凶星：** 雖凶但有節制，能轉為紀律 / 力量。
- **派外凶星：** 凶性更強，容易帶來困難主題。

> 古典核心。Sect 一個概念能解釋為什麼兩個有相同 Saturn 位置的人，一人覺得 Saturn 是嚴峻老師、另一人覺得是無情壓迫者。

### Step 2：整體骨架 — 元素 / 模式 / 半球分布

清點 10 顆主行星 + ASC + MC（共 12 個點）的元素 / 模式分布。

#### 元素分布
- **主導元素** = 最多 → 命主能量主色
- **缺失元素** = 0 個或 1 個 → 命主需有意識培養 / 補償的領域

詮釋例：
- 6 火 / 4 風 / 1 土 / 1 水 → 行動思考過剩，缺穩定與情感深度
- 0 火 / 5 土 / 4 水 / 3 風 → 內向沉重，缺乏自我啟動的火能

#### 模式分布
- 開創多 → 啟動力強、半途多
- 固定多 → 深耕但僵
- 變動多 → 適應強但不穩

#### 半球分布
- 行星集中**地平線上**（7-12 宮） → 公共領域、外向
- 行星集中**地平線下**（1-6 宮） → 私人領域、內向
- 集中**東半**（10, 11, 12, 1, 2, 3 宮） → 自我驅動
- 集中**西半**（4, 5, 6, 7, 8, 9 宮） → 他者驅動

#### 形態 (Chart Shape) — Marc Edmund Jones 系統
| 形態 | 描述 | 詮釋 |
|------|------|------|
| Bundle 束狀 | 全部行星在連續 120° 內 | 高度專注、有限視野 |
| Bowl 碗狀 | 全部行星在連續 180° 內 | 半球專精 |
| Bucket 桶狀 | Bowl + 一顆對立的「把手」 | 有獨立焦點推動 |
| Locomotive 火車狀 | 全部行星在連續 240° 內 | 強勁推進力、有領導感 |
| Splay 散開狀 | 三或更多分散群 | 多元、難整合 |
| Splash 噴濺狀 | 平均分散 | 廣泛興趣、議題多元 |
| See-saw 蹺蹺板 | 兩組分別集中、互對 | 二元主題、需平衡 |

### Step 3：三軸 — 日 / 月 / 升

這是命盤最常被簡化但最核心的三點：

#### Sun 太陽 — 「我想成為誰」
- 星座 = 我表達自我的風格
- 宮位 = 我尋找自我意義的領域
- 相位 = 影響自我意識展現的關鍵動力

#### Moon 月亮 — 「我本能反應 / 需要什麼才安全」
- 星座 = 情緒運作風格
- 宮位 = 情感安全的場域
- 相位 = 童年印記的影響來源

#### Ascendant 上升 — 「我如何進入世界」
- 星座 = 人格面具、外顯風格
- ASC ruler 所在 = 命主人生重心的「執行者」位置

> **三軸經典練習：** 把日 / 月 / 升各寫一個比喻形象，然後思考三者如何相處。例：
> 太陽射手 = 「探險家」、月亮天蠍 = 「深淵守護者」、上升處女 = 「精細工匠」 → 一個「精細工匠的外表，內在帶著深淵感受，靈魂渴望遠方」的綜合人。

### Step 4：上升守護星與守護鏈

#### Ascendant Ruler / Chart Ruler
- 守護上升星座的行星 = 命主的「靈魂執行人」。
- 它的位置（星座 + 宮位 + 相位 + 狀態）告訴你**命主人生的整體運行方向**。

例：
- 上升金牛，Venus 為 chart ruler，落 9 宮射手 → 命主一生重心在「擴展視野、出走、追尋意義」。
- 上升摩羯，Saturn 為 chart ruler，落 12 宮射手 → 命主結構與責任會在隱藏 / 靈性 / 退修領域展現。

#### Dispositor Tree（定位星鏈）
追蹤每顆行星的定位星：
1. 一顆行星「在某星座」
2. 該星座的守護星 = 它的定位星
3. 該定位星又在某星座 → 它有自己的定位星
4. 一直追到 final dispositor 為止（一顆行星位於自己守護的星座）。

**Final Dispositor 的意義：**
- 該行星整合了大部分行星的能量。
- 它所代表的議題是命主整盤的「總管」。

例：若 Sun in Leo（自定）、其他所有行星都最終追到 Sun → Sun 是終極 dispositor，命主整體生命被「自我表達 / 創造性」這個主題統合。

### Step 5：相位主題分析

#### Step 5.1 — 列出主要相位
- 主相位 + 容許度內次相位
- 標記 applying / separating
- 標記精確度（< 1° 命運性、1–3° 主題、3–5° 現出、> 5° 隱性）

#### Step 5.2 — 找相位圖形
- T-square、Grand Cross、Yod、Grand Trine、Kite、Mystic Rectangle、Stellium
- 圖形是命盤的「結構性簽名」

#### Step 5.3 — 分類整理
- **命主自我（Sun / Moon / ASC / chart ruler）** 的相位 — 性格主軸
- **凶星（Mars, Saturn, Pluto）** 的相位 — 困難議題、需整合處
- **吉星（Venus, Jupiter）** 的相位 — 資源、天賦、舒適區
- **代溝行星（外行星 Uranus / Neptune / Pluto）** 與內行星的相位 — 個人與時代議題的交匯

#### Step 5.4 — 主題凝練
從相位中抽出 3–5 個主軸主題（不是條列每個相位）：
- 例：「自我表達 vs 結構責任的張力」（Sun □ Saturn）
- 「親密深度的渴望與恐懼」（Venus ☍ Pluto）
- 「擴張理想 vs 現實限制」（Jupiter □ Saturn）

### Step 6：敏感點與業力主題

#### 月交點軸 (NN/SN)
- 演化派核心步驟。
- 五點檢視：SN 星座 + 宮位 + 守護星位置；NN 星座 + 宮位 + 守護星位置；交點軸的相位。
- 讀出「過去帶來什麼、這一世要走向什麼」。

#### Chiron
- 核心傷與療癒入口。
- 與軸線 / 三軸有相位 → 影響重大。

#### Lilith
- 被壓抑的、需要解放的能量。
- 多用於現代 / 演化派詮釋。

#### Part of Fortune
- 古典「自然順遂」的領域。
- POF 守護星位置 = 帶來該安康的執行人。

### Step 7：時間定位 — 當下流年

把命盤從靜態變成動態：

#### 啟動目前的行運 (Current Transits)
- 外行星行運（Saturn / Uranus / Neptune / Pluto）合 / 衝 / 方本命行星 → 當期主軸。
- 木星行運 → 該領域擴張機會。
- 火星行運 → 短期行動推力。

#### 二次推運 Moon (Progressed Moon)
- 約 27.3 年走完一周，每約 2.5 年換一個推運星座 / 宮位。
- 是「當期情緒主題的連續訊號」。

#### Profected Year（古典年運）
- 細節見 `08`。每年從 ASC 開始往下移一宮，該宮主星 = 該年的 Lord of the Year。

#### 整合
把當前流年放進命盤主題敘事：
「命主這幾年（Saturn 行運合本命 Sun in 7th）正在把『自我表達』與『關係結構責任』整合 — 這呼應命主出生時 Sun □ Saturn 的核心議題，是該議題第一次大型結構化的時刻。」

## 3. 解盤輸出建議結構

向命主呈現解盤結果時，建議結構：

```
1. 整體骨架 (Chart Overview)
   - Sect、元素 / 模式分布、半球分布、整體調性
   - 一句話總結這張盤的「形狀」

2. 三軸與核心人格
   - Sun / Moon / ASC 各自意義
   - 三者如何相處（協同 / 矛盾）

3. 上升守護與人生方向
   - Chart Ruler 的位置
   - 命主整體生命的「方向感」

4. 主軸議題（3-5 個）
   - 從相位、stellium、配置中抽出最核心的主題
   - 每個主題說：起源（哪個指標）+ 表現（如何在生活中展現）+ 整合方向

5. 敏感點主題
   - 月交點軸（過去 → 未來）
   - Chiron（傷與療癒）
   - Lilith（解放議題，視需要）

6. 當期流年定位
   - 目前主行運
   - 推運月亮
   - Profected year（古典年運）

7. 整合敘事
   - 把以上點編織成一個有節奏的故事
   - 不是條列符號，是一個可閱讀的人生敘事

8. 提問與對話
   - 開放式問題讓命主自我驗證
   - 留白給命主補充自己的經驗
```

## 4. 常見錯誤與避免

| 錯誤 | 為何錯 | 正確做法 |
|------|-------|---------|
| 把每個相位都讀成獨立段落 | 命盤是有機整體，分割就死 | 找主題，把相位編織成主題的論據 |
| 用「你的金星在天蠍所以你性慾強」的式句 | 簡化、刻板、命定論 | 「金星天蠍 → 渴望深度親密、易把關係視為轉化場域、可能對輕鬆關係感到不滿」|
| 忽略 Sect | 同樣行星位置在不同人盤上意義迥異 | 解盤先判 sect |
| 預測「會發生什麼」 | 占星不是這樣用的 | 預測「什麼主題會被啟動」 |
| 不提派別差異 | 給命主錯誤的「真理感」 | 標註「依希臘化派的看法…」「依現代心理派…」 |
| 無視 chart ruler | 漏掉命主整體軸線 | 永遠看 chart ruler 位置 |
| 全部用負面語氣 | 占星不是命算負債 | 任何凶星 / 凶相位都有整合方向 |
| 全部用正面語氣（雞湯式） | 失去深度真誠 | 該說的陰影要說，伴隨整合可能 |

## 5. 不同學派解盤側重的差異

### Hellenistic 古典 / 希臘化的解盤順序
1. Sect 派系
2. Ascendant Ruler 整體勢頭
3. Lights（日月）位置與狀態
4. Lord of Year（Profected）— 當年主題
5. Time-lord（ZR、firdaria）
6. Lots（POF, Spirit, Eros 等）
7. 行星整體 dignities + accidental dignities

### Modern Psychological 現代心理派的解盤順序
1. Sun-Moon-Ascendant 三軸
2. 元素 / 模式分布
3. Chart shape
4. 主要相位圖形（T-square, Grand Trine 等）
5. Outer planet aspects（Uranus / Neptune / Pluto 與內行星）
6. Chiron
7. 推運月亮 / 當前 transits 啟動的主題

### Evolutionary 演化派的解盤順序
1. Pluto 位置 + 宮位 + 相位（核心軸線）
2. 月交點軸（過去 → 未來）
3. Pluto 守護星與月交點守護星的位置
4. 演化主題（Pluto 與 NN 之間的關係）
5. 其他主要動力作為演化過程中的工具
6. 當期 transits 對 Pluto / 月交點的觸發

> **本 skill 預設整合三派：** 用古典骨架（sect、chart ruler、dignities）做技術判定；用心理派語言做議題敘事；用演化派視角做靈魂方向。如使用者明確偏好某一派，按其偏好調整。
