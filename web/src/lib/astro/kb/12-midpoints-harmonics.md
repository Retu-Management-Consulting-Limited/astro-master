# 12 — 中點與諧波 Midpoints and Harmonics

兩個進階分析技術，提供命盤的「結構性深層脈絡」。

## 1. Midpoints 中點

**中點 = 兩顆行星的中間位置。** 例：Sun 在 10° Aries、Moon 在 10° Cancer，則 Sun-Moon 中點 = 10° Taurus（取較短的弧）。

### 中點的本質

**中點理論的核心信念：** 兩顆行星之間的「精確中點」對應一個敏感點。當第三顆行星（或軸線、或行運星）合相 / 對沖此中點時，會啟動兩端行星的整合主題。

**詮釋公式：** 「[行星 A] = [行星 B] / [行星 C]」 表示行星 A 在 B 與 C 的中點上 → A 整合 BC 的主題。

### 中點的計算

**短中點（直接中點）：** 兩行星在黃道上的較短弧距的中央。
- A 在 0° Aries (0°)、B 在 60° Gemini (60°) → 短中點 = 30° Taurus
- A 在 20° Aries (20°)、B 在 20° Sagittarius (260°) → 短弧 = 120°，較短中點 = 80° = 20° Gemini

**長中點 / 反中點 (Antiscia 反映點)：** 短中點的 180° 對立點（也是該兩行星的另一個中點，從另一條弧計算）。

> **多數現代軟體（Solar Fire, Astro Gold）會自動列出每對行星的兩個中點。**

### 中點的解讀公式

中點被表示為 `A = B/C`（讀作 "A 等於 B 與 C 的中點"）：

| 中點配對 | 主題 |
|---------|------|
| Sun = Moon / Saturn | 自我意志在情感 / 結構之間整合；可能感到「父權壓力」與「母性需求」的張力 |
| Venus = Mars / Saturn | 性慾與責任的整合；承諾關係中的「禁慾與激情」拉扯 |
| Mercury = Mars / Pluto | 思維 / 言語進入「強烈、戰鬥性、深度」的領域 — 可能是辯論家、研究員、心理治療師 |
| MC = Sun / Pluto | 公開角色帶有「轉化命運感」 — 命主在公共領域有權力 / 影響力主題 |
| ASC = Moon / Neptune | 自我形象帶有「夢幻、共感、藝術氣質」 |

> **解讀關鍵：** 不必檢查所有可能中點（45 對行星！），只需檢查 **個人關鍵點（Sun, Moon, ASC, MC）對應的中點主題**。

### 漢堡學派 (Hamburg School) 的中點樹

**Alfred Witte (1920s) 創立。** 把每顆行星可能的中點全部列出，形成「中點樹 (midpoint tree)」 — 顯示該行星整合了哪些行星對的主題。

**中點樹的容許度：** 通常 1°–1.5°（極窄）。

**詮釋方法：**
1. 列出某顆行星的所有中點配對（例：Sun = Mercury/Venus, Sun = Mars/Jupiter, Sun = Saturn/Uranus 等）
2. 找最緊密的 3–5 個（< 1°）
3. 詮釋此行星「整合了哪些主題」

> **實務：** 漢堡學派的詳細應用需要專門軟體 / 手動計算。本 skill 簡介核心邏輯。

### Cosmobiology 宇宙生物學（Reinhold Ebertin）

德國 Ebertin (1950s) 把漢堡學派的中點系統與「90° 盤」結合，建立 **Cosmobiology** 流派：
- 用「90° 圖」(90° Dial) — 把整個 360° 黃道壓縮到 90° 上
- 在 90° 盤上，conjunction、square、opposition 都重合 — 這些「硬相位」最被強調
- 中點與硬相位的整合是核心解讀工具

**Ebertin《行星組合 (The Combination of Stellar Influences)》** 是本領域聖經：列出每對行星中點的具體詮釋。

### 中點的實務應用

#### 用於本命解讀
- 看 Sun, Moon, ASC, MC 各自在哪些中點上 → 揭示命主整合的主題
- 例：「Sun = Saturn/Pluto 中點 1° 內」→ 命主核心生命驅力含「結構 + 深度轉化」的整合議題（常見於有深度權力使命或重大轉化命運的人）

#### 用於行運預測
- 行運星合相 / 對沖某中點時 → 啟動該中點的兩端主題
- 例：行運 Saturn 合本命 Mars/Pluto 中點 → 啟動「行動 + 深度權力」的結構化議題（可能是重大決策、對抗、權力鬥爭）

#### 用於 synastry
- A 的行星合 B 的某個中點 → 啟動 B 該中點的主題
- 兩人共同的中點配對 → 命運性連結

## 2. Harmonics 諧波

**John Addey (1970s) 創立諧波占星 (Harmonic Astrology)。** 借用音樂中的「諧波 (overtones)」概念：每個整數頻率對應一個獨立的「次盤」，揭示主盤的某個結構面向。

### 諧波的本質

**核心邏輯：** 把每顆行星的位置乘以 N（諧波數），結果模 360° → 得到該行星在「N 諧波盤」中的位置。

**例：** Sun 在 10° Aries（= 10°）。在第 5 諧波盤中：5 × 10° = 50° → Sun in 5th harmonic 在 20° Taurus。

**意義：** 在第 N 諧波盤中，原本 360°/N 的相位（如 5 諧波對應 72° quintile）變成「合相」 — 因此 5 諧波盤揭示原本由 quintile / biquintile 串連的行星們的「合相結構」。

### 重要諧波及其對應

| 諧波 N | 360°/N | 對應相位（主盤）| 揭示的主題 |
|-------|--------|---------------|----------|
| 1 | 360° | 全盤 | 主盤本身 |
| 2 | 180° | Opposition | 兩極 / 對立 / 整合 |
| 3 | 120° | Trine | 流動 / 天賦 / 喜樂 |
| 4 | 90° | Square | 行動 / 危機 / 結構 |
| 5 | 72° | Quintile | 創造性才華 / 獨特模式 |
| 6 | 60° | Sextile | 機會 / 連結 |
| 7 | 51.4° | Septile | 命運 / 神秘 / 靈性使命 |
| 8 | 45° | Semi-square | 內在摩擦 / 努力 |
| 9 | 40° | Novile | 完成 / 整合 / 靈性圓滿 |
| 11 | 32.7° | — | 試煉 / 弱點 |
| 12 | 30° | Semi-sextile | 細節 / 適應 |

### 第 5 諧波（H5）— 才華盤

**最廣被使用的諧波盤之一。** 揭示命主的獨特創造模式 / 天才。

**做法：** 把所有本命行星位置乘以 5、模 360°，得到 H5 盤。
- H5 中相互合相的行星 = 主盤中以 quintile / biquintile 串連的行星
- H5 上升 / MC = 主盤中與 ASC / MC 形成 quintile 系列的行星 → 呈現命主的天才表達主題

### 第 7 諧波（H7）— 命運 / 靈性盤

**Septile 與「命運感」相關。** H7 揭示「靈魂使命 / 命運性連結」。
- H7 中合相主要行星 = 命主有命運感的能量配置
- H7 ASC 揭示「進入命運」的入口

### 第 9 諧波（H9）— 完成 / 婚姻盤

**Novile 與「圓滿、結合」相關。** 印度占星的「Navamsa（D9 盤）」與西方 H9 在計算上等同 — 都是 9 諧波。

**印度傳統：** D9 盤是「婚姻盤」、「靈性命運盤」。
- D9 中的 Sun, Moon, Venus 位置決定婚姻品質與時機
- D9 中的 ASC 揭示靈性方向

**西方：** Addey 系統把 H9 用於整體完成度、靈性發展、深度喜悅。

### 諧波分析的方法論

#### Step 1：建立諧波盤
用占星軟體建立 H5、H7、H9 諧波盤（或用印度占星軟體建立 D9）。

#### Step 2：詮釋方法
- 把諧波盤當作「次盤」獨立讀
- 注意 H5 / H7 / H9 中的合相 = 主盤中對應相位的「壓縮版」
- H 盤上升 / MC 是揭示該諧波主題的關鍵點

#### Step 3：與主盤交叉印證
- 主盤的天賦相位 (quintile) → 在 H5 變合相 → 強烈才華主題
- 主盤的命運相位 (septile) → 在 H7 變合相 → 靈魂使命主題

### 諧波盤的限制

1. **計算精確性敏感：** 出生時間每差 4 分鐘，諧波盤可能差距很大（因為位置乘以 N，誤差也乘以 N）。
2. **不適合本命解讀的「主軸」。** 諧波盤是補充，不是替代。
3. **詮釋仍在發展中。** 漢堡學派 / Cosmobiology / 諧波都是 20 世紀才系統化的，與古典 / 心理派的整合不夠成熟。

## 3. 中點與諧波的整合應用

### 場景：尋找天才 / 獨特才華
- 主盤：找 quintile（72°）/ biquintile（144°）相位
- H5：看 H5 ASC 與哪顆行星合相 → 才華的「執行人」
- 中點：看 Sun, Moon, MC 在哪些中點上 → 整合的天才模式

### 場景：尋找命運性主題 / 靈魂使命
- 主盤：找 septile 系列相位
- H7：看 H7 ASC 與哪顆行星合相 → 命運入口
- 中點：看 Sun = Pluto/Neptune 等深度中點

### 場景：婚姻 / 親密關係深度分析
- 主盤：7 宮、Venus、Mars、Juno、Lots
- H9 / D9：婚姻盤主題
- 中點：Venus = Mars/Saturn（責任之愛）、Venus = Jupiter/Saturn（關係的擴張與限制）等

## 4. 派別差異

| 學派 | 立場 |
|------|------|
| 古典 / 希臘化 | 不使用（時代不存在）|
| 漢堡學派 | 中點為主軸 — 整套系統建立在中點上 |
| Cosmobiology (Ebertin) | 中點 + 90° 盤為主 |
| 諧波學派 (Addey) | 諧波盤為主 |
| 主流現代心理派 | 部分採用，多用主盤為基礎 + 諧波為補充 |
| 演化派 | 較少採用，但會看 H9 作為靈魂方向參考 |

> **本 skill 預設：** 主盤分析為主；中點與諧波在以下情況啟動：
> - 命主明確問「我有什麼天賦 / 才華」 → H5 + 中點輔助
> - 命主明確問「我的命運使命」 → H7 + 中點
> - 命主明確問「婚姻深度」 → H9 / D9 + Venus / Mars 中點
> - 解盤遇到主盤難以解釋的主題 → 用中點補強
