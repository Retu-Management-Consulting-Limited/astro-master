# 15 — 現代心理占星 Modern Psychological Astrology

現代心理占星把占星與深度心理學（特別是榮格學派）結合，把命盤視為「心靈的地圖」。**興起於 1970–80s 的英國，由 Liz Greene、Howard Sasportas 為核心。**

## 1. 學派起源

### 主要奠基者

- **Dane Rudhyar (1895–1985)：** 美國法裔，《The Astrology of Personality》(1936)。把占星從「事件預測」轉向「人格發展」，提出「人本占星 (Humanistic Astrology)」 — 現代心理占星的奠基。
- **Liz Greene (1946– )：** 英國占星家、榮格分析師，《Saturn: A New Look at an Old Devil》(1976) 開啟新時代。倫敦 Centre for Psychological Astrology (CPA, 1983) 共同創辦人。
- **Howard Sasportas (1948–1992)：** 美裔英籍，《The Twelve Houses》(1985)。專注宮位心理學詮釋。CPA 共同創辦人。
- **Stephen Arroyo：** 《Astrology, Psychology and the Four Elements》— 把元素心理學系統化。
- **Robert Hand：** 《Planets in Transit》(1976)、《Planets in Composite》— 行運與合盤心理學詮釋。
- **Erin Sullivan：** 《Saturn in Transit》、《Dynamics of the Unconscious》。
- **Glennys Lawton (et al)：** 後 CPA 一代。

### 與古典的根本差別

| 議題 | 古典 | 現代心理 |
|------|------|---------|
| 命盤是什麼 | 命運的天文寫照 | 心靈結構的象徵地圖 |
| 預測什麼 | 客觀事件 | 心理主題的啟動 |
| 行星是什麼 | 帶來事件的客觀力量 | 內在心理動力 / 原型 |
| 凶星 | 客觀的不利因素 | 待整合的陰影 |
| 諮詢方法 | 占星師告知命主 | 占星師與命主對話、探索 |
| 個案目標 | 知曉命運 / 趨吉避凶 | 心理整合、自我認識 |

## 2. 榮格學派的核心概念在占星上的應用

### 2.1 原型 (Archetypes)

**榮格：「集體無意識中存在普遍的心靈模式 — 原型 — 由所有人類分享。」** 行星 = 原型的化身。

| 行星 | 主要原型 |
|------|---------|
| Sun | 自性 (Self)、英雄 (Hero)、王 (King) |
| Moon | 母親 (Mother)、滋養者 (Nurturer)、家 |
| Mercury | 信使 (Messenger)、騙術者 (Trickster)、煉金術士 |
| Venus | 愛人 (Lover)、女神 (Aphrodite)、價值持有者 |
| Mars | 戰士 (Warrior)、開拓者 (Pioneer) |
| Jupiter | 智者 (Sage)、王者 (King)、教師 |
| Saturn | 老人 (Senex)、權威 (Authority)、限制者 / 教師 |
| Uranus | Prometheus、革命者、覺醒者 |
| Neptune | 神秘者 (Mystic)、藝術家、犧牲者 |
| Pluto | Hades、轉化者、深淵 |
| Chiron | 受傷療癒者 (Wounded Healer) |

### 2.2 個體化過程 (Individuation Process)

**榮格：「個體化 = 整合無意識內容，成為獨特的『自性』。」**

占星上對應的工作：
- **Persona（人格面具）= ASC** → 整合外顯與內在的張力
- **Shadow（陰影）= 凶星 / 困難相位** → 整合被排斥的特質
- **Anima/Animus = Venus / Mars** → 整合內在的另一性別能量
- **Self（自性）= Sun + 整盤的整合** → 達成完整的個體性

### 2.3 陰影工作 (Shadow Work)

**陰影 = 個體無意識中被壓抑、不被認可的部分。**

占星上的陰影指標：
- **凶星 + 困難相位**：Mars / Saturn / Pluto 的 square / opposition
- **困難宮位的行星**：8 宮、12 宮的行星
- **南交點的特質**（演化派也用此）
- **Lilith**（被壓抑的狂野能量）

**陰影工作 = 把這些「被排斥的」能量重新整合為意識的一部分。**

### 2.4 投射 (Projection)

**投射 = 把自己無意識的特質投射到他人身上。**

占星的投射工具：
- **DSC（下降點）= 我們把 ASC 不認的部分投射到伴侶 / 對手身上**
- **Synastry**：強烈相位常與「投射動力」相關 — 對方展現我們否認的能量
- **困難 transits**：當 transits 啟動本命陰影時，常見「對方變成我」的投射觸發

## 3. Liz Greene 的核心貢獻

### 3.1 重塑 Saturn

**《Saturn: A New Look at an Old Devil》(1976) 是現代心理占星的轉捩點。**

**古典：** Saturn 是大凶星、限制、苦痛之源。
**Liz Greene：** Saturn 不是要打垮你，是要塑造你 — 透過時間、限制、現實檢驗，讓你發展出深度與成熟。

**Saturn 的兩面：**
- **陰影：** 自我批判、恐懼、僵化、權威崇拜、抑鬱
- **整合：** 紀律、責任、長期視野、智慧、權威感

**Saturn Return（28-30 歲）的重新詮釋：**
- 不是「災難」 — 是命主從「青春假象」進入「成人責任」的儀式性轉化
- 該整合什麼？該結束什麼？該認真投入什麼？

### 3.2 神話與占星的整合

Liz Greene 把希臘神話深度運用到占星詮釋：
- 《The Astrology of Fate》(1984)：用神話原型詮釋命運感
- 《Relating: An Astrological Guide to Living with Others》：關係占星的心理學深化
- 《The Outer Planets and Their Cycles》：外行星的世代心理學

### 3.3 神話原型的詮釋

每顆行星 / 主題 / 圖形對應的神話：

| 占星元素 | 神話原型 |
|---------|---------|
| Saturn | Cronus（時間之神，吞噬子女）、Senex（老人原型）|
| Mars | Ares（戰爭）、Hephaestus（鍛造工匠）|
| Pluto | Hades（地下王）|
| Chiron | Centaur Chiron（受傷的療癒者）|
| Venus | Aphrodite（性愛）/ Demeter（穀物）|
| Lilith | Adam 第一妻（被父權放逐的女性）|
| Mercury | Hermes（信使、騙術者、靈魂引導者）|
| Jupiter | Zeus（王、雷電、家父權）|
| Neptune | Poseidon（海洋）/ Dionysus（迷醉）|
| Uranus | Prometheus（盜火者，反叛諸神）|

## 4. Howard Sasportas 的宮位心理學

**《The Twelve Houses》(1985)** 把每個宮位深度心理學化：

### 宮位的「童年議題」鏡映

每個宮位都對應一個發展階段的議題：
- **1 宮：** 自我感的建立（嬰兒早期，「我是誰」）
- **2 宮：** 安全感與價值（早期飲食 / 物質滿足）
- **3 宮：** 語言與基礎連結（學齡前的環境探索）
- **4 宮：** 家與根（家庭印記）
- **5 宮：** 創造性表達（玩、被看見）
- **6 宮：** 任務與服務（學校 / 工作早期）
- **7 宮：** 一對一連結（伴侶 / 鏡像）
- **8 宮：** 親密深度（青少年期的性、權力、危機）
- **9 宮：** 信念與意義（成人開始建立世界觀）
- **10 宮：** 社會角色（成人的事業）
- **11 宮：** 集體歸屬（社群、未來理想）
- **12 宮：** 消融與超越（中老年的靈性整合 / 退休）

### 行星進入宮位 = 該心理階段的議題被啟動

例：Saturn in 4th = 童年家的結構性壓抑（嚴父或缺席的情感支持） → 成年的「家」議題帶 Saturn 主題（建構穩定家結構的渴望、易感到家是責任而非舒適）

## 5. Stephen Arroyo 的元素 / 模式心理學

### 元素失衡 → 心理特質

#### 火元素 Fire
- **多：** 行動驅力強、外向、自信但躁動 / 自我中心
- **少：** 缺乏自我啟動、自信不足、需要透過外界刺激才動

#### 土元素 Earth
- **多：** 務實、穩定、可靠但僵化 / 物質主義
- **少：** 不接地、計畫難落實、不穩定的物質感

#### 風元素 Air
- **多：** 思考社交強、抽象能力高，但脫離身體與情感
- **少：** 表達 / 概念化困難、難客觀化問題

#### 水元素 Water
- **多：** 情感豐富共感強，但易被情緒淹沒、邊界模糊
- **少：** 情感切離、共感弱、需練習情緒覺察

### 模式失衡

#### 開創 Cardinal
- **多：** 啟動力強、半途多
- **少：** 啟動困難

#### 固定 Fixed
- **多：** 持久執著、僵化
- **少：** 缺乏耐力 / 中心軸

#### 變動 Mutable
- **多：** 適應靈活、不穩定
- **少：** 不易應變、機械化

### Arroyo 的整合方法
- 缺乏的元素 / 模式需要**有意識培養**
- 透過伴侶、生活情境、專注練習補強

## 6. Erin Sullivan 與「動態無意識」

**《Dynamics of the Unconscious》、《Saturn in Transit》。**

核心觀念：占星行運是「無意識動力浮現的時間結構」。

### 行運的心理學意義

| 行運 | 心理學主題 |
|------|----------|
| Saturn 對 Sun 的硬相位 | 自我意識 vs 結構 / 父權的整合 |
| Saturn 對 Moon | 童年情感印記的再檢視 |
| Saturn Return | 第一次成人結構建立 |
| Pluto 對 Sun | 自我的死亡 - 重生 |
| Pluto 對 Venus | 親密 / 愛 / 價值的根本轉化 |
| Uranus 對 Sun | 自我的突破與覺醒 |
| Neptune 對 Sun | 自我邊界的消融、靈性開啟或迷霧 |

## 7. 心理派的解盤方法論

### 與古典的差異
- **不評分行星「好壞」**
- **不下事件預測**
- **使用「議題」、「主題」、「課題」語言而非「會發生」**
- **強調命主的選擇性與成長空間**
- **重視諮詢過程的對話**

### 心理派解盤的標準結構
1. 整體骨架（元素 / 模式 / 半球分布）
2. Sun-Moon-Ascendant 三軸
3. 主要相位主題（用心理學語言重述每個相位的議題）
4. Outer planet 的世代主題（Uranus / Neptune / Pluto 與內行星的相位）
5. Chiron 與療癒議題
6. 當前 transits 啟動的主題

### 提問風格
- 「你會如何描述自己面對 [X 主題] 的感覺？」
- 「在你生命中，[X 領域] 是怎樣的故事？」
- 「這個相位對應一個張力 — 你怎麼處理它？」

避免：
- 「你會 / 你必然 / 你逃不過」
- 「你的命運是…」
- 「這是凶象，要小心」

## 8. 心理派的優勢與限制

### 優勢
- **整合性強：** 把命盤變成「自我探索的工具」而非「命運審判」
- **諮詢倫理高：** 命主有選擇權與主體性
- **與心理治療 / 自我成長兼容**
- **避免迷信化、宿命化**

### 限制
- **可能過度抽象化：** 心理主題說了等於沒說（「你有自我整合的議題」 — 誰沒有？）
- **不擅長預測具體事件：** 命主問「我什麼時候會結婚」 — 心理派常無法給出時間結構
- **可能忽略客觀指標：** 缺乏 sect、dignities 等技術判定，可能把弱位置浪漫化
- **「個人成長語言」的疲勞：** 部分使用者會覺得心理派太「自我成長雞湯」

## 9. 與其他學派的整合

### 與古典的互補
- 古典提供結構與時間預測；心理派提供深度詮釋
- **理想：** 古典 sect + dignities → 客觀指標；心理派語言 → 深度敘事

### 與演化派的差異
- 心理派：把命盤當作「這一世的心靈地圖」
- 演化派：把命盤當作「靈魂多生累積的演化路徑」
- 心理派不必接受輪迴前提；演化派的核心是輪迴

### 在本 skill 中的角色

**主敘事語言：** 大部分解盤敘事用心理派語言（議題、課題、整合、陰影、主題）。

**技術判定：** 用古典補強（sect, dignities, time-lord）。

**靈魂方向：** 用演化派補強（月交點、Pluto 軸線）。

> **三派整合，不二選一。**
