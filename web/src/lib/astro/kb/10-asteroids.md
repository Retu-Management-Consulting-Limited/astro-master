# 10 — 主要小行星與虛點 Asteroids and Hypothetical Points

本章涵蓋本命盤詮釋常用的小行星與虛點。**這些不是傳統占星核心，但廣泛被現代占星師（特別是女性主義占星師、心理派、演化派）採用，補充七古典 + 三外行星的「人類經驗光譜」。**

## 0. 小行星的歷史背景

- **1801–1807：** Ceres, Pallas, Juno, Vesta 是最早被發現的四顆小行星，發現於小行星帶。
- **1919–現在：** 發現的小行星超過百萬顆，大多無占星意義。
- **占星界 1970s–1980s：** Demetra George、Eleanor Bach 等女性占星師把 Ceres / Pallas / Juno / Vesta 引入主流，作為「Sun-Moon-Mercury-Venus-Mars 四男神 vs 四女神」的補充。

## 1. 四大主要小行星 (Big Four Asteroids)

這四顆是現代占星最廣泛使用的小行星。**符號與位置在多數占星軟體 (AstroSeek, Astro.com, Solar Fire) 都有預設。**

### ⚳ Ceres 穀神星

**神話：** 羅馬農業女神（希臘 Demeter）。其女兒 Persephone 被冥王劫持後，Ceres 哀痛使大地不收，終於談判 Persephone 半年回地面、半年留冥府 → 季節循環之神話起源。

**核心象徵：**
- **無條件滋養 / 母性的源頭**（與月亮的「個人母親」不同；Ceres 是滋養本身的原型）
- **失去與哀悼**（Persephone 失蹤的傷）
- **食物 / 飲食 / 餵養**（很常與飲食習慣、飲食障礙、共有食物的儀式相關）
- **大地與季節循環**
- **照顧 vs 控制**（Ceres 對女兒的愛中含控制成分）

**關鍵問題：** 我如何被滋養、又如何滋養他人？我的「失去」核心傷在哪？

**身體 / 議題對應：** 飲食障礙（神經性厭食 / 暴食常與 Ceres 有關）、母女關係、失去親人的哀悼、慢性照顧議題。

### ⚴ Pallas Athena 智神星

**神話：** 希臘智慧戰爭女神 Athena，從 Zeus 頭中誕生（不經產道）。是工藝、戰略、智慧、政治、藝術的女神。

**核心象徵：**
- **創造性智慧 / 模式辨識**
- **戰略 / 政治**
- **藝術與工藝**
- **「父親之女」原型**（從父親頭中誕生 → 與父權結盟、與女性原則疏離）
- **獨立自主、不結婚的女性力量**

**關鍵問題：** 我如何運用智慧解決問題？我的創造性策略在哪展現？

**強位置 / 議題：** 強 Pallas 常見於設計師、政治家、律師、棋手、藝術家、研究者。

### ⚵ Juno 婚神星

**神話：** 羅馬婚姻女神（希臘 Hera），Zeus 之妻。多次被丈夫背叛但仍堅持婚姻 — 婚姻女神也是「被背叛的妻子」原型。

**核心象徵：**
- **承諾關係 / 婚姻**（vs Venus 的浪漫吸引）
- **平等夥伴關係的渴望**
- **背叛、嫉妒、權力鬥爭**（陰影面）
- **靈魂伴侶 / 命定關係**
- **婚姻中的「我們」**

**關鍵問題：** 我在承諾關係中需要什麼？我的「靈魂伴侶」期待是什麼？我如何處理背叛主題？

**Synastry：** Juno 連結（特別是 Juno 合相 Sun / Venus / Mars / Moon）常見於命定感重的伴侶關係。

### ⚶ Vesta 灶神星

**神話：** 羅馬灶神（希臘 Hestia），守護家中爐火的處女神。其神廟由「Vestal Virgins」（誓守童貞的女祭司）守護。

**核心象徵：**
- **聖火 / 神聖專注**
- **獨身 / 獨處的神聖性**（不必然「禁慾」 — 是「不為他人 / 為使命專注」）
- **熱情焦點 / 專業使命**
- **性與靈性的整合**（神廟妓女傳統中 Vesta 也與性儀式相關）

**關鍵問題：** 我為什麼神聖之事「燃燒」？我的中心使命在哪？我如何處理獨處與親密的張力？

**強位置：** 常見於僧侶、深度研究者、單身藝術家、無孩終身投入志業者命盤。

## 2. 性與愛主題的小行星

### 433 Eros 愛神星

**神話：** 希臘愛慾之神（羅馬 Cupid）。原始性慾與激情的化身。

**核心象徵：**
- **原始慾望 / 性吸引**
- **「無法理性控制的渴望」**
- **創造性激情**

**詮釋：** Eros 在哪個星座 / 宮位 → 命主激烈渴望什麼。Synastry 中 Eros 連結強烈時 → 強烈性吸引。

### 763 Cupido 邱比特

**核心象徵：** 浪漫愛、家族團結、藝術、追求美。
較少使用，但在愛情主題盤中有意義。

### 16 Psyche 賽姬

**神話：** Eros 的妻子，凡人女性。經歷四個任務後成神。象徵「靈魂與愛的整合過程」。

**核心象徵：** 靈魂深度連結、心靈成長透過愛、心理深度。

**Synastry：** Psyche 合 Eros = 靈魂與激情的合一；命運感重的關係。

### 80 Sappho 莎芙

**神話：** 古希臘女詩人 Sappho，居住於 Lesbos 島，寫女性間的愛 → 「lesbian」一詞的源頭。

**核心象徵：** 同性愛、女性之間的深度連結、詩意、藝術之愛、不被主流接納的愛。

### 1221 Amor 愛慕

**核心象徵：** 慈悲之愛、無私的愛、靈性的愛（與 Eros 的肉體之愛對比）。

## 3. 創傷 / 療癒 / 命運主題的小行星

### 2 Pallas（已上）— 智慧之傷的整合

### 1181 Lilith（小行星 Lilith，與黑月 Mean/True Lilith 不同！）
- **不要混淆：** 黑月 Lilith 是月亮軌道的遠地點（虛點）；小行星 1181 Lilith 是真實小行星。**它們的位置完全不同，意義也不同。**
- 小行星 Lilith：被排斥的女性力量、不被馴服、邊緣性。

### 8 Flora 花神
- 花、香、優雅、性、創造性表達

### 24 Themis 法律之母
- 公正、法律、長期道義、宇宙秩序

### 56 Melete 練習 / 沉思
- 專注練習、藝術修行

### 944 Hidalgo（半人馬族）
- 政治反抗、為信念奮戰、邊緣性領袖

### 5145 Pholus（半人馬族）
- 「小事引發大改變」、業力小傷大果

### 7066 Nessus（半人馬族）
- 邊界違反、虐待、權力濫用、循環性傷害

> **半人馬族 (Centaurs)：** 軌道介於 Saturn 與 Pluto / Neptune 之間的小行星類別。Chiron 是其中一員。其他重要的有 Pholus, Nessus, Asbolus, Chariklo, Hylonome — 各有獨特的「邊緣 / 命運性轉化」象徵。

## 5. Trans-Neptunian / Hypothetical Points 海王外 / 虛點

### Hamburg School / Uranian Astrology 的 8 顆 trans-Neptunian
漢堡學派 (Alfred Witte, 1920s) 提出的 8 顆「假設行星」（不是真實天體）：

| 名稱 | 象徵 |
|------|------|
| Cupido | 結婚 / 家庭 / 藝術 |
| Hades | 過去 / 退化 / 隱藏的 |
| Zeus | 創造性能量 / 火 / 領導 |
| Kronos | 權威 / 高度 / 政府 |
| Apollon | 擴張 / 多重 / 商業 |
| Admetos | 凝聚 / 深度 / 死亡 |
| Vulcanus | 強大力量 / 巨大 |
| Poseidon | 靈性 / 啟示 / 思想 |

> **限制：** 這些不是真實天體，計算依賴特定軟體（Solar Fire, Witte 規範）。漢堡學派以外的占星家較少採用。

### Eris 厄里斯
- **真實天體**（2003 年發現的矮行星，比冥王還大）。
- **神話：** 不和女神，「金蘋果之爭」的引爆者。
- **占星意義：** 不被邀請的部分、抗爭、女性憤怒、揭穿不公。
- 周期約 561 年，幾乎不在生命中移動 — 主要用整體位置詮釋世代議題。

### Sedna 賽德娜
- 2003 年發現，極遠的矮行星（軌道約 11,000 年）。
- **神話：** 因紐特海洋女神，被父親背叛淹死後成為深海之母。
- **占星意義：** 深層背叛、無法用言語表達的傷、原住民議題、海洋與生態。

## 6. 解讀小行星的方法論

### 步驟
1. **先解完十顆主行星**（古典 7 + 現代 3）— 命盤主結構由它們決定。
2. **看小行星「合相」哪些主結構（< 3°）** — 例：Ceres 合 Sun → Ceres 主題進入核心人格；Juno 合 7 宮起點 → 婚姻是命主重要議題。
3. **小行星本身的星座 / 宮位** — 進階補充（< 1° 緊密合相時最強）。
4. **不要過度詮釋小行星之間的相位** — 容易資訊爆炸而失去主軸。

### 詮釋優先順序
1. 與本命三軸（日 / 月 / 升）合相的小行星 — 最強影響
2. 與本命四軸（ASC, MC, IC, DSC）合相的小行星
3. 與本命主行星合相 / 對沖的小行星
4. 落在重要宮位（5, 7, 8, 10）的小行星
5. 其他位置 — 細節補充

### 實際應用情境
- **婚姻 / 親密議題：** Juno + Eros + Psyche
- **滋養 / 飲食 / 母性：** Ceres
- **使命 / 專注：** Vesta
- **智慧策略：** Pallas
- **同性愛主題：** Sappho
- **創傷議題：** Chiron + Nessus + Pholus

## 7. 不同學派對小行星的立場

| 學派 | 立場 |
|------|------|
| 嚴格古典 / 希臘化 | 不使用（時代不存在）|
| 中世紀古典 | 部分用 Lots 補充類似主題 |
| 現代心理派 | 部分用，特別是 Big Four |
| 演化派 | 廣泛採用，特別是 Chiron / 半人馬族 |
| 女性主義占星 | 核心使用 — Big Four 是修正父權偏向的工具 |

> **本 skill 預設：** 主要用 Big Four (Ceres, Pallas, Juno, Vesta) + Chiron。其他小行星視具體議題啟用。不會把小行星設為解盤主軸。
