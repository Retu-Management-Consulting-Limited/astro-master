# 08 — 推運與行運 Transits, Progressions, Time-Lords

預測技術的目標是回答：「**這個主題什麼時候會被啟動 / 高峰 / 結束？**」 占星不預測「事件」，預測「主題的時間結構」。

## 0. 預測技術的整體架構

```
時間預測技術
├── 行運 (Transits) — 真實天體當下位置 vs 本命盤
├── 推運 (Progressions)
│   ├── 二次推運 (Secondary Progressions, "1 day = 1 year")
│   ├── 太陽弧 (Solar Arc Directions)
│   └── 主限向運 (Primary Directions, 古典)
├── 太陽 / 月亮返照 (Solar / Lunar Returns)
├── 古典 Time-Lord 系統
│   ├── Annual Profections (年運)
│   ├── Zodiacal Releasing (黃道釋放)
│   ├── Firdaria (波斯主限)
│   └── Decennials (十年期)
└── 食 (Eclipses) — 重大事件預測點
```

**現代主流：** 行運 + 二次推運（+ 太陽弧）為主，太陽返照為輔。
**古典系統：** Annual Profections + Zodiacal Releasing 為核心。
**完整解讀：** 多技術交叉印證才可靠 — 單一技術看到的事件，需要其他技術同步啟動才強烈顯化。

## 1. Transits 行運

**行運 = 真實天體當下的位置。** 行運觸碰本命盤行星 / 軸線時，啟動相應主題。

### 行運速度與重要性

| 行運星 | 速度 | 影響特性 | 持續時間 |
|-------|------|---------|---------|
| Pluto | 0.004°/日 | 深度結構轉化 | 一個合相主題持續 1.5–3 年 |
| Neptune | 0.006°/日 | 消融 / 迷霧 / 靈性 | 1–2 年 |
| Uranus | 0.011°/日 | 突破 / 革命 | 1 年 |
| Saturn | 0.034°/日 | 結構責任 / 限制 | 6–9 個月 |
| Jupiter | 0.083°/日 | 擴張 / 機會 | 2–4 個月 |
| Mars | 0.5°/日 | 行動 / 衝突 | 數天到 2 個月 |
| Sun / Mercury / Venus | 1°/日 | 短期色彩 | 數日 |
| Moon | 13°/日 | 即時情緒 | 數小時 |

### 解讀行運的優先順序

#### 高優先（一定要看）
1. **Saturn / Uranus / Neptune / Pluto 行運** 觸碰本命行星 / 軸線 / 月交點 — 主要人生階段事件
2. **Saturn / Pluto 行運** 觸碰個人光體（Sun, Moon）或 chart ruler — 重大轉折
3. **Jupiter 行運**合本命 Sun 或 chart ruler — 機會 / 擴張時期

#### 中優先
4. **行運合本命四軸（ASC, MC, IC, DSC）** — 軸線觸碰永遠重要
5. **行運合本命月交點** — 命運啟動
6. **食 (Solar / Lunar Eclipse)** 觸碰本命行星或軸線（< 3°） — 大事件

#### 低優先
7. 內行星行運（Sun, Mercury, Venus, Mars） — 提供「色彩」與「日常啟動」，不單獨產生大事件
8. 行運 Mars 在某宮 — 該領域短期行動推力

### 行運的「精確 - 高峰 - 出相位」三階段

對外行星（Saturn 以外）行運的單次合相通常會經歷三個時點：
1. **First hit (applying)：** 行星精確合本命點
2. **Retrograde hit：** 行星逆行回來再次精確合（「重來一次」、處理深度議題）
3. **Direct hit (separating)：** 順行後第三次合，確認 / 終結

**Saturn 通常會 2–3 次精確合 (hits)，Uranus / Neptune / Pluto 會 3 次甚至 5 次。**
**第一次與最後一次中間的時間差就是「該主題的工作期」。**

### 行運的詮釋公式
「[行運星之動力] 透過 [相位] 啟動 [本命星之動力]，在 [本命宮位場域] 顯化 — [質地：擴張/危機/突破/消融/重建]」

範例：
- **Transit Saturn ☌ Natal Sun in 4th house**
- → 「Saturn 的結構 / 責任壓力 (Saturn) 透過合相 (☌) 啟動了命主的核心自我 (Sun)，在家 / 根 / 心理深處 (4th) 場域顯化 — 是命主重新建立『我是誰』與家族 / 根的關係的結構性時期，可能伴隨家庭責任增加、父親議題、搬家、生活根基重整。」

## 2. Secondary Progressions 二次推運

**核心邏輯：** 「**出生後 X 天 = 出生後 X 年**」。
- 出生後第 1 天的天體位置 = 命主 1 歲時的「內在進化狀態」
- 出生後第 30 天的天體位置 = 命主 30 歲時的內在進化狀態

**意義：** 不是預測外部事件，而是顯示**命主內在心理 / 性格的演化**。

### 推運速度（依本命行星速度按比例）
| 行星 | 出生後天 - 年比例 | 在生命中走完一周 |
|------|-----------------|----------------|
| Sun | 1°/日 = 1°/年 | 360 年（沒有人活到一周）|
| Moon | 13°/日 = 13°/年 | 約 27.3 年 |
| Mercury / Venus | 約 1°/年 | 360 年 |
| Mars | 約 0.5°/年 | 720 年 |
| 外行星 | 緩慢，多數人一生內變化不大 | — |

### 最重要的兩個推運
#### Progressed Sun
- 約每 30 年換一個推運星座
- 大型生命階段轉換（例：30 歲推運 Sun 由獅子進處女 → 從外放 / 創造主題轉向精修 / 服務主題）

#### Progressed Moon（最重要的推運點）
- 每約 2.5 年換一個推運宮位
- 每約 27.3 年完成一周（與本命月亮形成完整循環，類似 Saturn Return 的「月亮回歸」）
- **詮釋：** 每階段的情感主題、內在情緒週期
- **Progressed Moon 與本命月亮的合相（27.3 年週期）：** 「Moon Progression Return」— 重大情感階段過渡

### 推運相位
推運行星之間 / 推運行星與本命行星形成的相位也都重要：
- **Progressed Sun ☌ Natal Saturn：** 內在自我與結構 / 限制的整合期
- **Progressed Moon ☌ Natal Pluto：** 情感深度轉化期

### 推運盤的繪製
- 取「出生後第 X 天（X = 命主年齡）」的天體位置
- 用本命的軸線 / 宮位框架（不重新計算 ASC / MC，宮位用本命的）
- 看推運行星「落在本命的哪個宮位」+「推運行星與本命行星的相位」

## 3. Solar Arc Directions 太陽弧推運

**核心邏輯：** 把所有本命行星「整體前移命主年齡 × 太陽 1 年走的度數」（約 1° 每年）。

### 與二次推運的差異
- 二次推運：每顆行星按自己速度推進
- 太陽弧：所有行星統一以 Sun 的速度推進（每年約 1°）
- 結果：Solar Arc 推運讓**慢速行星也能在生命中產生顯著相位**（不像二次推運中外行星幾乎不動）

### 解讀重點
- 每年所有行星都前進約 1° → 軸線（ASC, MC）也前進約 1°/年
- 適合看「人生中某個時點，本命某行星 / 軸線會精確接觸到另一本命行星」
- **Solar Arc directions 與 outer planet aspects 在生命中的「精確化時點」是大事件預測重點。**

例：本命 Sun 與 Saturn 相距 30°（半六分），則命主約 30 歲時，Solar Arc Sun 會精確合 Natal Saturn → 重大成熟 / 結構性轉折。

### 計算簡化版
- 本命任兩顆行星之間的角距離 ÷ 1°/年 ≈ 該相位精確化的年齡（適用於太陽弧；緊密相位則更早顯化）

## 4. Solar / Lunar Returns 太陽 / 月亮返照盤

### Solar Return 太陽返照盤
- **時點：** 每年生日當天太陽精確回到本命太陽位置的瞬間
- **意義：** 該年度的「年盤」 — 從這次生日到下次生日的整體主題
- **計算地點：** 一般用命主當年居住地（部分占星師主張用出生地，有爭議）
- **解讀：** 像讀一張小型本命盤；特別注意 SR 的 ASC、SR 行星落本命宮位

### Lunar Return 月亮返照盤
- **時點：** 月亮每約 27.3 天回到本命月亮位置
- **意義：** 該月情緒主題 / 短期事件
- **較少做為主流預測，但對當月情緒節奏判定有用**

### 進階：Demi Returns
- **半返照 (Demi Return)：** Sun / Moon 行運到對立本命位置 → 該年度 / 該月「中段轉折點」

## 5. Hellenistic Time-Lord 系統

希臘化占星核心預測技術。**每個時間點都有一個「主管行星 (time-lord)」決定那段時間的主題質地。**

### 5.1 Annual Profections 年運推進

**核心：** 每年生日後，從 ASC 開始，每年活化一個宮位，按順時針？— 不對，是按黃道順序（順時針 vs 逆時針：黃道 1→2→3→…→12 是占星上的順序）。

#### 計算
- 0–1 歲：1 宮（從 ASC 所在星座開始）
- 1–2 歲：2 宮
- 2–3 歲：3 宮
- …
- 11–12 歲：12 宮
- 12–13 歲：回到 1 宮（重複循環，每 12 年一個 cycle）
- 12, 24, 36, 48, 60, 72, 84 歲都是「1 宮年」

#### Lord of the Year (年主星)
**= 當年活化宮位的星座的守護星**

例：
- ASC 在牡羊
- 命主 27 歲 → 27 ÷ 12 = 2 餘 3 → 處於第三循環的第 4 個位置 = 4 宮年
- 4 宮起點是巨蟹（從牡羊起算第 4 個星座）
- 巨蟹守護星是 Moon → **Moon = 27 歲那年的 Lord of the Year**

#### Lord of the Year 的詮釋
1. 看 Lord of the Year 在本命盤中的位置 → 該年「事件發生的領域」
2. 看 Lord of the Year 的本命狀態 → 該年事件的整體質地（順 / 逆）
3. 看當年 transits 觸發 Lord of the Year → 高峰時點
4. 看 Lord of the Year 在當年的位置（progressed / Solar Return）
5. 看當年活化宮位（profected house）內的本命行星與軸線

### 5.2 Zodiacal Releasing (ZR) 黃道釋放

**最強的古典 time-lord 技術，由 Vettius Valens 傳承下來。** 將命主一生劃分為「章節 / 段落 / 句子」，每個層級對應一個 time-lord。

#### 兩個主要起點 (Lots)
- **Lot of Spirit (起點)：** 看**事業 / 命運主動性 / 公開人生** 的章節
- **Lot of Fortune (起點)：** 看**身體 / 物質 / 機運 / 被動命運** 的章節

> **公式：**
> - Lot of Spirit (Diurnal): `ASC + Sun - Moon`；(Nocturnal): `ASC + Moon - Sun`
> - Lot of Fortune (Diurnal): `ASC + Moon - Sun`；(Nocturnal): `ASC + Sun - Moon`

#### 四個層級 (Four Levels)
| 層級 | 比喻 | 持續時間 | 用途 |
|------|------|---------|------|
| L1 | 章節 (Chapter) | 數年到二十年 | 人生大階段 |
| L2 | 頁面 (Page) | 數月到數年 | 階段內的子主題 |
| L3 | 段落 (Paragraph) | 數週到數月 | 具體月份主題 |
| L4 | 句子 (Sentence) | 數日 | 具體日 |

#### 各 L1 階段的長度（依星座守護星）

每個 L1 階段的時長 = 「啟動星座」的守護星的「最小年數 (least years)」週期：

| 星座 | 守護星 | L1 長度（年）|
|------|-------|-------------|
| 巨蟹 / 獅子 | Moon / Sun | 25 / 19 |
| 雙子 / 處女 | Mercury | 20 |
| 金牛 / 天秤 | Venus | 8 |
| 牡羊 / 天蠍 | Mars | 15 |
| 射手 / 雙魚 | Jupiter | 12 |
| 摩羯 / 水瓶 | Saturn | 27 |

> 例：若 Lot of Spirit 在獅子 → 第一個 L1 階段持續 19 年，由 Sun 主管。19 歲後進入下一個星座（處女），由 Mercury 主管 20 年。

#### 高峰期 (Peak Periods)

某些段落是「高峰」(peak periods)，事件密度最大：
- **Major Peak 最大高峰：** L1 = 與 Lot of Fortune 同星座，或與 Lot of Fortune 對應的「上中天」(angular from Fortune) 位置
- **Moderate Peak：** 對沖 Lot of Fortune 的星座
- **Minor Peak：** 與 Lot of Fortune 形成 inferior square 的星座
- 在 Major Peak 期間，ZR 顯示命主人生最重大的事件

#### Loosing of the Bond 鬆綁
- 當 ZR 由某個段落跳到「對宮的下一個段落」時，會發生 "loosing of the bond" — 一個重大的內在轉折，常標記人生方向的徹底改變。

### 5.3 Firdaria 波斯主限

中世紀波斯占星的時主系統：
- 一生分成幾個大段，每段由一顆行星主管
- 大段內又細分為更小的子段
- **日盤順序：** Sun (10) → Venus (8) → Mercury (13) → Moon (9) → Saturn (11) → Jupiter (12) → Mars (7) → Nodes (3+2)
- **夜盤順序：** Moon → Saturn → Jupiter → Mars → Nodes → Sun → Venus → Mercury

### 5.4 Decennials 十年期

每顆行星輪流主管 10 年 9 個月（依特定順序），是中世紀 / 文藝復興時期的時主系統，現代較少用。

## 6. 食 (Eclipses) 預測

食 = Sun 與 Moon 在月交點附近的合相 (日食) / 對沖 (月食)。

### 食的占星意義
- **比一般新月 / 滿月強 10–100 倍**
- 觸碰本命行星 / 軸線（< 3°）時 → 重大事件
- 影響期：通常從食前 6 個月到食後 6 個月，主要事件常落在食前後 1–3 個月
- 食的影響可延伸 3 年以上（特別是與外行星觸發配合時）

### Saros Cycle 莎羅週期
- 每 18.6 年食週期循環一次（因月交點週期）
- 命主 18.6 歲、37 歲、55.8 歲時的食點與本命食點同位 → 個人命運循環的關鍵年齡

### Pre-Natal Eclipse 預出生食
- 命主出生前最近一次的食點
- 行運觸發此點 → 命運性大事件
- 部分演化派視為「靈魂進入這一世的業力簽名」

## 7. 預測解讀的整合方法

### Step 1：列出當期所有啟動因素
```
[當前年齡 X] 啟動：
- Transits（外行星觸碰本命）：
- Progressed Moon 在 X 宮 X 星座
- Progressed Sun 與本命 X 行星形成 X 相位
- Solar Arc 推運：
- Solar Return 重點配置：
- Profected Year — Lord of the Year = X
- ZR L1 段落 = X 星座，主管 = X 行星；L2 段落 = X
- 最近食點：
```

### Step 2：找重複主題
不同技術指向同一行星 / 主題 → 該主題會強烈顯化。

### Step 3：時間定位
- 哪個月 / 季度是事件高峰？（看 transit 精確化時點）
- 第一次 / 重來 / 終結三階段的時間表

### Step 4：詮釋整合
不是「會發生 X 事」，而是「X 主題會強烈被啟動，命主會在這段時間經歷 / 處理這個主題」。

## 8. 預測技術的限制與倫理

1. **占星不能預測命運的「具體形式」。** 同一個 Saturn 行運合 Sun，一個人經歷工作壓力、另一人面對父親病、第三人決定創業 — 都是同一主題的不同顯化。
2. **不要說「你會 / 你必須」。** 說「這段時間的主題是 X，可能透過 [幾種典型形式] 顯化」。
3. **不要嚇唬命主。** Saturn / Pluto / Eclipse 觸碰個人點不必然「壞事」，是「結構性轉化期」。
4. **預測涉及健康、死亡、重大決策時，** 必須加上保留 — 占星不取代醫療 / 法律專業。
5. **避免「自我實現預言」。** 過度告知命主某時點會有 X 事，可能讓命主朝該方向潛意識行動。
