# 05 — 敏感點 Sensitive Points

本章涵蓋四軸 (Angles)、月交點 (Lunar Nodes)、Lilith、幸運點 (Part of Fortune)、Vertex 等命盤上不是行星但極為重要的點。

## 1. 四軸 (The Four Angles)

**四軸是命盤中最敏感的點。** 行星合相四軸時影響力被極度放大；行運觸碰四軸時帶來重大事件。

### Ascendant (ASC) 上升點
- **位置：** 1 宮起點，東方地平線升起的黃道度數。
- **核心：** 自我形象、人格面具、身體、進入世界的方式、第一印象、生命軸線。
- **古典：** 「靈魂之點」、生命之始；Ascendant Lord（守護上升星座的行星）= 命主整體的代表星。
- **現代：** ego 顯現、自我與世界的介面。
- **演化：** 這一世選擇的人格載體。
- **計算精確性敏感：** 出生時間每差 4 分鐘 → ASC 約差 1 度。

### Descendant (DSC) 下降點
- **位置：** 7 宮起點，西方地平線落下的黃道度數，與 ASC 對沖 180°。
- **核心：** 一對一關係、伴侶投射、公開敵人、合夥、他者鏡映。
- **詮釋：** 我們把 ASC 不認的部分投射到 DSC 上，伴侶 / 對手往往展現我們未能整合的特質。

### Medium Coeli (MC) 中天 / 天頂
- **位置：** 10 宮起點，當地正午時太陽會經過的點，也是黃道與當地子午線的交點。
- **核心：** 公開形象、事業 / 志業、社會角色、最高成就方向、人生方向。
- **古典：** 「事業之宮起點」、當下命主在世間的權威表現。
- **詮釋：** MC 與 ASC 不必然成 90°（除非用 Equal House 等）— 它們是兩個獨立的天文計算點。

### Imum Coeli (IC) 天底
- **位置：** 4 宮起點，當地正午太陽的對面點，與 MC 對沖 180°。
- **核心：** 家、根、心理深處、童年印記、私人空間、晚年歸所。
- **詮釋：** IC = 「來自何處」、MC = 「往何處去」 — 兩端是命主人生的縱軸。

### 軸線詮釋
- **ASC-DSC 軸：** 「我」與「他者」的對話軸。
- **MC-IC 軸：** 「公開」與「私人」的對話軸；「未來方向」與「根 / 起點」的縱軸。
- 兩軸交叉形成命盤的「十字」骨架。
- **行星合相四軸（5° 內）影響極大：**
  - 合 ASC：強烈塑造自我形象與身體
  - 合 DSC：透過伴侶 / 對手浮現
  - 合 MC：定義事業與公開角色
  - 合 IC：影響家、根、心理深處

### Anti-Vertex（反 Vertex）/ Equatorial Ascendant
- **Equatorial Ascendant (East Point)：** 在赤道上升起的黃道點，與 ASC 不同。
- 較少用，與 ASC 重複度高。

## 2. 月交點 (Lunar Nodes)

**月交點是月球軌道與黃道交點。** 北交點 (North Node, NN, ☊) 是月球從黃道南向北穿越的點；南交點 (South Node, SN, ☋) 在對面，為從北向南穿越的點。**永遠 180° 對分。**

### 兩種計算
- **真交點 True Node：** 即時計算，會有逆行 / 留 / 短期順行的微動。
- **平均交點 Mean Node：** 平均化的軌道計算，永遠逆行。
- **本 skill 預設用 True Node**（多數現代占星師的選擇），但兩者差距通常 < 2°。

### 移動週期
- 約 **18.6 年** 繞黃道一周。
- 永遠逆行（除真節點偶爾微小順行）。
- 每 1.5 年換一個星座 / 對軸。

### 月交點的本質詮釋

#### South Node (SN) — 過去軸
- **古典：** Caput Draconis 的對宮 (Cauda Draconis 龍尾) — 古典視為「漏失點」、能量不留之處，類凶。
- **現代心理：** 已熟悉、舒適但已不滋養的模式；童年 / 家族傳承的本能反應；天賦但易耽溺。
- **演化派（核心）：** 累世帶來的記憶與才能、靈魂的「老路」；舒適但若停留會阻礙成長。

#### North Node (NN) — 未來軸
- **古典：** Caput Draconis (龍頭) — 古典視為「能量匯聚點」、類吉。
- **現代心理：** 必須有意識培養的領域、新的成長方向、初期會感覺笨拙陌生。
- **演化派（核心）：** 這一世靈魂選擇要前往的演化方向、未開發的整合領域。

### 解讀方法論

**演化派 (JWG) 月交點解讀的「五點檢視」：**
1. **南交點所在星座** — 過去帶來的人格傾向與議題
2. **南交點所在宮位** — 過去經驗發生的場域
3. **南交點守護星 (NN/SN ruler) 所在位置與相位** — 把過去議題具象化
4. **北交點所在星座、宮位、守護星** — 演化方向的詳細座標
5. **與月交點軸線形成相位的行星** — 影響演化過程的關鍵動力

### 月交點與行運
- **Nodal Return（月交點回歸）：** 約每 18.6 年（月交點回到本命位置）— 重大演化里程碑。
- **Nodal Reversal（半週）：** 約每 9.3 年（月交點到本命的對位）— 常與「方向反轉」事件相關。
- **行運合相 NN：** 該領域加速擴展、命運開展。
- **行運合相 SN：** 舊主題被檢視 / 釋放、業力結算。
- **食 (Eclipses)：** 太陽 / 月亮在月交點附近的合相 / 對沖 — 大事件、命運轉折點。

## 3. Lilith 莉莉斯（黑月 Black Moon）

**Lilith 並非實體天體，是月球橢圓軌道的「遠地點 (apogee)」。** 命盤上代表女性的、未被馴化的、被排斥的、原始狂野的部分（不限性別）。

### 三種 Lilith
| 計算法 | 名稱 | 特性 |
|--------|------|------|
| **Mean Lilith** | 平均黑月 | 數學平均化、平滑、最常用 |
| **True / Osculating Lilith** | 真實黑月 | 即時計算、會有顯著震盪 |
| **Asteroid 1181 Lilith** | 小行星莉莉斯 | 真實小行星，不同於黑月概念 |

> **本 skill 預設用 Mean Lilith**（最廣泛使用版本）。若使用者特別指定，可用 True 或小行星版。

### 詮釋核心
- **神話原型：** 莉莉斯是亞當的第一任妻子（猶太神話），拒絕屈從而被放逐 — 不被父權系統容納的女性力量。
- **盤上意義：** 命主被社會 / 童年禁止的「狂野」、無法馴化的慾望、被父權壓抑的部分。
- **正向：** 自主、性能量真實、不為討好他人而活。
- **陰影：** 報復性、極端拒絕、自我邊緣化、把他人推開。

### Lilith 在各星座 / 宮位

**星座：** 顯示哪個能量原型被壓抑 / 需要解放（例：Lilith in Leo = 對被看見的渴望被壓抑；Lilith in 12th = 整個禁忌領域被掩埋在無意識）。

**宮位：** 顯示哪個生活領域是命主的「狂野邊界」。

### 解讀提醒
- Lilith 的詮釋在現代仍在發展，沒有像古典行星那樣的標準化系統。
- 解讀時宜問：「在哪個領域，你感受過『不被允許做自己』？」
- 演化派詮釋常與性、創傷、解構父權有關。

## 4. Part of Fortune 幸運點（Fortuna ⊗）

**Part of Fortune** 是阿拉伯點 (Arabic Parts / Lots) 中最重要的一個。古典占星核心元素之一，現代多數占星師也會看。

### 計算公式
古典系統依日 / 夜盤分別計算（**極重要，常被忽略**）：

- **日盤 (Diurnal, 太陽在地平線上)：**
  `POF = ASC + Moon - Sun`
- **夜盤 (Nocturnal, 太陽在地平線下)：**
  `POF = ASC + Sun - Moon`

> 現代軟體預設常常不分日 / 夜（一律用日盤公式），這是錯誤。**正確使用必須區分。**

### 詮釋
- **古典：** 「身體 / 物質生命的福樂之處」— 命主的身體 / 物質繁榮 / 生計來源容易達成的領域。
- **現代：** 自然安康感、對命主而言「事情自然會順」的領域 / 主題。
- **靈性 / 演化派：** 內在喜悅的入口、命主與生命之流對齊的點。

### POF 的星座 / 宮位
- 星座 = POF 體驗的風格
- 宮位 = POF 顯化的領域
- 與 POF 守護星（其所在星座的守護）的位置 → 「實現命主之福的執行人」

### 其他重要阿拉伯點 (Lots)

| 名稱 | 公式（日盤）| 詮釋 |
|------|-----------|------|
| Lot of Spirit | ASC + Sun - Moon（夜盤反） | 心智 / 命運自主性、職涯方向（Hellenistic 用於職業） |
| Lot of Eros | ASC + Venus - Spirit | 慾望、愛的對象 |
| Lot of Necessity | ASC + Fortune - Mercury | 命運性的限制、無法迴避之處 |
| Lot of Courage | ASC + Mars - Fortune | 主張勇氣、戰鬥領域 |
| Lot of Victory | ASC + Jupiter - Spirit | 成功與勝利之處 |
| Lot of Nemesis | ASC + Fortune - Saturn | 隱藏敵人、遺失之物 |
| Lot of Marriage | ASC + DSC - Venus（男性） / ASC + Venus - Saturn（女性） | 古典婚姻時機 |

> 阿拉伯點是希臘化 - 阿拉伯傳統的核心技術。本 skill 在預測 / 卜卦時會啟用，本命解讀以 POF 為主。

## 5. Vertex 與 Anti-Vertex

**Vertex** 是黃道與「主垂直圈 (prime vertical)」的西側交點。**Anti-Vertex** 在對面，東側。

### 詮釋
- **Vertex（通常落在 5–8 宮間）：** 「命運性相遇之點」、不可預測的因緣、被動性的命運安排。
- **Anti-Vertex：** 主動發起命運的點。
- **行運合相 Vertex：** 常與重要相遇 / 突發事件相關（婚姻、職業轉折、命定關係）。

### 使用注意
- Vertex 並非主流必看點，但在 synastry 與重要事件預測中有用。
- 常用於分析「為什麼某人 / 事件感覺如此命定」。
- 西方占星界對 Vertex 的詮釋仍有分歧。

## 6. 行星「自己的點」(Joys 喜悅之宮)

希臘化占星每顆古典行星都有一個「喜悅」的宮位 (joy)，行星在該宮位特別能展現其象徵：

| 行星 | Joy in House |
|------|------------|
| Mercury | 1st |
| Moon | 3rd |
| Venus | 5th |
| Mars | 6th |
| Sun | 9th |
| Jupiter | 11th |
| Saturn | 12th |

> 這不是現代主流必用，但在希臘化分析中重要 — 例如 Saturn in 12th 雖然 12 宮古典凶，但 Saturn 在那裡「joyful」，能展現深度的內省、靈修、孤獨工作能力。

## 7. 其他敏感點

### Asteroid 點（4 主要小行星）
詳見 `10-asteroids.md`。Ceres / Pallas / Juno / Vesta 是廣泛被納入的女性原型補充。

### 預測點 (Prenatal Eclipse, 預出生食點)
- 命主出生前最近一次的食點（日食 / 月食）。
- 詮釋：靈魂進入這一世的「業力簽名」入口。
- 行運合相此點 → 命運性事件。

### Solar Equinox / Solstice 點（春分秋分夏冬至點）
- 0° Aries / 0° Cancer / 0° Libra / 0° Capricorn — 這些「世界軸 (world axis)」上的位置在世運盤 (mundane astrology) 中極重要。
- 本命行星合相世界軸 (3° 內) 暗示與時代 / 集體主題深度共振。

### Galactic Center 銀河中心
- 目前位於約射手 27°（會因歲差緩慢移動）。
- 本命行星合相 GC（< 2°）→ 與「超個人 / 集體訊息」的特殊連結，常見於藝術家、靈性導師、天才命盤。

## 8. 解讀優先順序

當綜合所有敏感點時，建議按以下順序處理（避免資訊過載）：

1. **四軸（ASC, MC, DSC, IC）** — 命盤骨架，必看。
2. **月交點軸（NN/SN）** — 演化主題的核心軸線，必看。
3. **Part of Fortune（按日 / 夜盤正確計算）** — 古典核心點，建議看。
4. **Lilith** — 視個人興趣加入；現代心理 / 演化解盤常用。
5. **Vertex** — 重要關係 / 事件分析時加入。
6. **其他 Lots、Joys、銀河中心** — 進階分析時使用。
