# 09 — 擇時與卜卦 Electional & Horary Astrology

兩種應用占星的「實務技術」：
- **擇時 (Electional)：** 「**何時**做某件事最好？」
- **卜卦 (Horary)：** 「對這個**具體問題**，答案是什麼？」

兩者都根植於古典占星，**現代 / 心理派多不深入；要做這兩個必須回到希臘化 / 中世紀古典規則。**

## 1. Electional Astrology 擇時占星

### 1.1 擇時的本質

**選一個「時刻」，使其本身的占星條件最有利於某個目的。** 這個時刻將成為事件的「出生時間」 — 事件的命盤決定事件的命運。

### 1.2 擇時的應用場景
- 婚禮日期 / 時間選擇
- 公司 / 產品發布 / 開幕
- 重要簽約、面試、報名
- 重大旅行出發
- 投資 / 開戶
- 醫療手術（傳統還有月相考量）
- 任何一次性的重要起點

### 1.3 擇時的核心規則

#### Rule 1：避開 Moon Void of Course (空亡月亮)

**Moon Void of Course (V/C)：** 月亮從目前所在星座離開、與其他行星沒有形成新主相位、直到月亮進入下一個星座為止的這段時間。

**詮釋：** 「事情不會發生 / 不會結出果實 / 像沒做過一樣」 — 古典強烈避開。

**例外：** Moon V/C 對「不希望事情有結果」的事項可以反向利用（如想拒絕但已被迫去面試？）。

#### Rule 2：選擇 Moon 強位置
- 月亮在巨蟹（入廟）、金牛（入旺 3°）→ 強
- 月亮在摩羯（陷）、天蠍（落 3°）→ 弱
- 月亮入相位向吉星 (Venus/Jupiter) → 加吉
- 月亮入相位向凶星 (Mars/Saturn) → 慎用
- **Waxing Moon (新月→滿月)：** 適合「成長 / 開始」事項
- **Waning Moon (滿月→新月)：** 適合「結束 / 削減」事項

#### Rule 3：強化主管行星 (Significator)

每件事有對應的「主管行星」：
| 事件類型 | 主管行星 |
|---------|---------|
| 婚禮 | Venus（女性主動） / Sun（男性主動）/ 7 宮主 |
| 開店 / 商業 | Mercury / Jupiter |
| 旅行 | Mercury（短）/ Jupiter（長）|
| 創業 | Sun + Jupiter / 10 宮主 |
| 簽約 | Mercury |
| 學習 | Mercury / Jupiter |
| 醫療手術 | Mars（外科）/ 6 宮 |

**做法：** 選擇時間使該主管行星處於：
- 強尊貴位置（in domicile / exaltation）
- 角宮位置（1, 4, 7, 10）
- 與 Moon 形成有利相位
- 順行（不逆行）
- 不焦傷（不在 Sun 8.5° 內）

#### Rule 4：強化相關宮位
- 婚禮 → 強化 7 宮 / 7 宮主
- 開店 → 強化 10 宮 / 10 宮主
- 出版 → 強化 9 宮 / 9 宮主
- 投資 → 強化 2 宮 / 8 宮、Jupiter

#### Rule 5：避開凶星在角宮 / 一宮
- 1 宮起點（ASC）合凶星（Mars / Saturn / Pluto）→ 強烈避開
- 4, 7, 10 宮的凶星也要謹慎
- 凶星在 6, 8, 12 宮影響相對小（古典：凶星「待在」凶宮反而是中和）

#### Rule 6：上升星座要與事件性質相容
- 牡羊上升 → 火、行動、開創 — 適合衝刺、競爭
- 金牛上升 → 穩定、財富、長期 — 適合金融、不動產
- 雙子上升 → 訊息、靈活 — 適合溝通、學習
- 巨蟹上升 → 家、情感、滋養 — 適合家庭、餐飲
- 獅子上升 → 創造、表現、王者 — 適合表演、藝術、領導
- 處女上升 → 精細、服務 — 適合技術、健康
- 天秤上升 → 關係、美感 — 適合婚禮、合夥
- 天蠍上升 → 深度、轉化、權力 — 適合心理、神秘、危機處理
- 射手上升 → 擴張、信念 — 適合出版、教學、長途旅行
- 摩羯上升 → 結構、長期、建構 — 適合企業、不動產
- 水瓶上升 → 創新、群體 — 適合科技、社運
- 雙魚上升 → 靈性、藝術 — 適合創作、靈修

### 1.4 擇時的流程

#### Step 1：明確事件類型與主管行星

#### Step 2：列出候選時段
（命主能執行的時間範圍）

#### Step 3：用占星軟體（如 Solar Fire, Astro Gold）逐一檢查每個候選時刻
- ASC 星座是否與事件性質相容
- Moon 是否有 V/C / 是否強 / 與凶星相位
- 主管行星是否強 / 順行 / 與 Moon 連結
- 凶星是否在 1, 4, 7, 10 宮起點 5° 內

#### Step 4：篩選「次優」時段
- 不可能找到完美時段（特別是要在數天內定的事項）
- 篩到「沒有大忌」+「主管行星狀態尚可」+「Moon 不空亡」即合用

#### Step 5：精細化
- 在優選時段內，精細到分鐘的選擇
- 看 ASC 度數的精確 — 越接近某星座中段越穩定

### 1.5 擇時的限制

1. **無法選太久遠的時段（命主必須能在那時段執行事件）**。
2. **某些事件本身有結構性問題的時候，再好的擇時也救不了**（婚姻雙方根本不合，擇好日子無用）。
3. **天文事件不可改變**（某月可能 Mercury Rx + Mars 在 1 宮 + 凶星觸發 — 整月難以擇出好時段）。
4. **古典規則來自前現代世界**（醫療手術的擇時規則不取代現代醫療判斷）。

### 1.6 重要的「不擇時」(Inception Charts)

當無法選擇時間（事件已被外部決定），可以分析事件的「inception chart」（事件啟動時的盤）來預測其結果。
- 公司成立日期、書籍出版日、戰爭爆發點 — 都可作為 inception chart 解讀。

## 2. Horary Astrology 卜卦占星

### 2.1 卜卦的本質

**卜卦的核心信念：** 一個誠懇的問題在被提出的那個時點，宇宙會在當時的天空中「投射」出答案 — 把問題提出當下的盤計算出來，按古典規則解讀，可得到對該問題的明確回答。

> **這是占星裡爭議最大的領域。** 現代心理派多不接受卜卦的客觀預測能力；古典派視卜卦為占星的最高應用之一。

### 2.2 卜卦的條件

#### 必要條件
1. **問題誠懇且重要**（不是隨便問玩的）
2. **問題明確具體**（「我會被錄取嗎？」可；「我未來會怎樣？」不可）
3. **命主真心想知道答案**（情緒投入是「問題能量」進入盤的條件）
4. **問題未事先被決定**（不是已知答案要驗證）
5. **問題只問一次**（同樣問題不能反覆問同樣占星師）

#### 不適合卜卦的問題
- 純好奇 / 玩笑性的問題
- 命主已決定答案、只想要驗證
- 涉及他人隱私且命主無正當關連的事
- 涉及醫療診斷 / 法律判決的取代性問題（必須警告命主求專業意見）

### 2.3 卜卦盤的設定

**時間：** 占星師完整理解問題的那個時刻（不是命主發送訊息的時刻）。
**地點：** 占星師當下的位置（不是命主的位置 — 這是傳統規則，現代有爭議）。
**宮位制：** **Regiomontanus 是古典標準；Placidus 也廣泛使用。Whole Sign 在卜卦傳統中極少用。**

### 2.4 卜卦的核心邏輯：Significator System

每個問題涉及兩方（或多方）。每方在盤中由一顆行星「代表」(significator)：
- **問者 (Querent)：** 1 宮主（ASC ruler）+ Moon
- **問事 (Quesited)：** 看問題類型決定是哪宮的主星

#### 問事 → 對應宮位
| 問題 | 對應宮位 |
|------|---------|
| 自己 / 健康 / 整體狀態 | 1 |
| 金錢 / 財產 / 動產 | 2 |
| 兄弟姊妹 / 短旅 / 鄰居 | 3 |
| 父母 / 家 / 不動產 | 4 |
| 戀愛 / 子女 / 投機 | 5 |
| 工作 / 健康 / 寵物 | 6 |
| 伴侶 / 公開敵人 / 合夥 | 7 |
| 死亡 / 配偶之財 / 共享資源 | 8 |
| 出國 / 高等教育 / 法律 | 9 |
| 事業 / 母親 / 公開角色 | 10 |
| 朋友 / 群體 / 希望 | 11 |
| 隱藏敵人 / 退隱 / 業力 | 12 |

> **派生宮位 (Derived Houses)：** 問題可能涉及第三方（例：「我朋友的兒子會考上嗎？」），這時要把該人的宮位作為「新的 1 宮」往後算 — 朋友是 11 宮，朋友的兒子 = 11 的 5 宮 = 命主盤的 3 宮（11 + 5 = 16, 16 - 12 = 4… 等等實際算法是「11 之第 5 宮」=計算複雜）。

### 2.5 卜卦的判斷規則

#### Rule 1：盤是否「可判讀」(Considerations Before Judgment)

古典規則檢查盤本身是否「適合給出答案」：
- **ASC 在 0–3° 或 27–30°：** 「盤太早 / 太晚」 — 問題未成熟 / 問題已過時，不適合判斷
- **Saturn 在 1 宮：** 占星師警告（自己可能誤判）
- **Saturn 在 7 宮：** 占星師可能不公正
- **Moon V/C：** 「事情不會結果」 — 強烈傾向 No
- **月在過渡 (in late degrees of a sign)：** 情境正在改變

> 古典 William Lilly 的這些「judgment 前檢查」在現代仍被使用，但要彈性運用。

#### Rule 2：兩個 significator 之間的相位

**相位形成過程預示事件發展：**

| 相位類型 | 預測 |
|---------|------|
| **Conjunction applying** (兩星即將合相) | 兩方將會結合 / 事情會發生 |
| **Sextile / Trine applying** | 順利、和諧地完成 |
| **Square / Opposition applying** | 困難重重，但若仍能完成意味著有衝突結果 |
| **No applying aspect** | 事情不會發生 |
| **Significator separating from each other** | 雙方正在分開、事情不結果 |

#### Rule 3：「Translation of Light 光線傳遞」與「Collection of Light 光線匯聚」

當兩個 significator 沒有直接相位但有「中介」時：
- **Translation：** 第三顆行星先離開一個 significator（出相位），然後即將合相另一個 → 「中介者把訊息傳遞，事情透過第三方促成」
- **Collection：** 兩個 significator 都向第三顆慢速行星形成相位 → 「事情透過某個權威 / 結構性人物收集後達成」

#### Rule 4：互容 (Mutual Reception) 與接收 (Reception)
- **Mutual Reception：** 兩星在彼此守護的星座 → 「雙方互相接受」、強烈助益
- **Reception by other dignity：** 透過 exaltation / triplicity / term / face 也算入

#### Rule 5：阻礙 (Frustration / Prohibition)
- 兩 significator 即將相位，但其中一方先與另一行星形成相位 → 那個行星「阻止」事情完成

#### Rule 6：障礙 (Refranation)
- 兩 significator 即將相位，但其中一方在精確相位前轉逆 (turns retrograde) → 「事情就要成功時被取消」

### 2.6 常見卜卦判讀

#### 「Will I get the job? 我會錄取嗎？」
- 1 宮主 = 命主；10 宮主 = 工作職位
- 兩星的相位：
  - applying conjunction / trine / sextile → Yes
  - applying square → 可能會錄取但有衝突
  - applying opposition → 可能但有對立
  - 完全無相位 / separating → No
- 額外：Moon 與兩星的關係加強判斷

#### 「Where is my lost object? 失物在哪？」
- 2 宮（自己的東西）的主星位置
- 該主星的星座 / 元素 → 提示物理位置（土 = 在地上 / 地下；火 = 在熱源附近；風 = 在高處 / 通風處；水 = 在水 / 液體附近）
- 該主星的宮位 → 進一步定位（4 宮 = 在家；3 宮 = 鄰居 / 車裡；6 宮 = 工作場所）

#### 「Will the relationship work out? 這段關係會成嗎？」
- 1 宮主（命主）+ 7 宮主（對方）
- Moon = 額外信號
- applying aspect → 關係將深化；無 applying aspect → 不結果

### 2.7 卜卦的限制與爭議

1. **卜卦是占星最有爭議的應用。** 古典派視為核心；現代心理派常質疑其客觀性。
2. **不應替代專業判斷。** 醫療、法律、投資的「答案」不能由卜卦決定。
3. **占星師主觀因素多。** 同一個盤不同占星師會有不同判讀。
4. **倫理：** 不應該為他人問同意問題（如「會他離開他妻子嗎？」 — 涉及第三方隱私與選擇權）。

## 3. 兩者的整合

### 擇時 + 卜卦的組合應用
- 用卜卦先確認「該不該做這件事」
- 卜卦顯示「會成」 → 用擇時找最佳執行時點
- 卜卦顯示「不會成」 → 重新評估或暫不做

### 與本命盤的關係
- 卜卦盤獨立於命主本命盤（不需要命主的本命資料）
- 擇時盤要與命主本命盤的關鍵點對齊（例：擇時的 ASC 應與本命某重要點和諧；不要在命主 Saturn 行運期間擇婚禮）

## 4. 派別與現代立場

| 議題 | 古典 / 希臘化 - 中世紀 | 現代心理 | 演化 |
|------|---------------------|--------|------|
| 卜卦合法性 | 正統技術 | 多數質疑 / 不練習 | 不太用，但承認其象徵性 |
| 擇時嚴格度 | 嚴格遵守規則 | 較彈性 / 簡化版 | 更看與本命的演化主題對齊 |
| 月空亡 | 強烈避開 | 視情況 | 不一定避開 |

> **本 skill 立場：** 對於擇時與卜卦，遵循古典規則而非現代簡化版。現代心理派的工具不適合這兩個應用 — 必須回到 William Lilly / Bonatti / Lilly's Christian Astrology 的傳統規則才能得到可靠結果。
