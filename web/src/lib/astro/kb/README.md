# 占星正典知识库 (Astrology Canon KB)

Reference knowledge base for grounding Molly's interpretations in established
astrological method — the "calibrated interpretation system" behind Track B.

- **Source**: distilled from the standalone 星盤大師 agent skill (Hellenistic/Traditional,
  Modern Psychological, Evolutionary). Western canon only.
- **Scope**: general astrological technique & method. **No PII.** Personal case
  files (individual charts/readings) and the advanced-techniques chapter (derived
  from one person's case data) are intentionally **excluded**.
- **Usage**: this is reference material, not code. Do **not** dump it wholesale
  into prompts (token cost). Extract the relevant method/section and inject
  targeted snippets alongside the deterministic facts from `dignities.ts`.
